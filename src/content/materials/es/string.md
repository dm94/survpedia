El cordel es un cordón fino fabricado retorciendo fibras en una hebra continua. Se utiliza para atar, sujetar y unir materiales cuando se necesita un conector flexible.

## Descripción de lo que es

El cordel es ligero y flexible, con una resistencia que depende del tipo de fibra y del número de cabos. Puede anudarse, envolverse y tensarse sin romperse si está bien fabricado.

## Origen y dónde encontrarlo

- **Entornos**: Allí donde existan fuentes de fibra (plantas, animales).
- **Signos**: Hebras retorcidas o trenzadas hechas de fibras o tendones.
- **Es un material procesado.**

### Procesado mínimo requerido

1.  **Recogida de fibras**: Fibras vegetales, pelo de animal o tendones.
2.  **Preparación**: Limpieza y alineación de las fibras.
3.  **Torsión**: Enrollar las fibras en una sola hebra.
4.  **Plegado**: Retorcer varias hebras juntas para darles resistencia.

### Herramientas necesarias para trabajar con él

- Manos o husos sencillos para retorcer.
- Ganchos o clavijas para tensar.
- Cuchillos para cortar las fibras.

### Formas comunes de uso

- **De un solo cabo**: Ataduras ligeras y fijaciones temporales.
- **De dos o tres cabos**: Cordón más fuerte para herramientas y costura.
- **Trenzado**: Trenzas planas o redondas para mayor durabilidad.

### Posibles sustitutos

- **[Yarn](/es/materials/yarn)**: Cordón de fibra más grueso y suave.
- **[Plant fibers](/es/materials/plant-fibers)**: Utilizadas directamente como cordaje.
- **[Leather](/es/materials/leather)**: Tiras de cuero crudo para fijaciones resistentes.

### Limitaciones y fallos comunes

- **Abrasión**: Las fibras se deshilachan al rozar con superficies rugosas.
- **Humedad**: Las fibras naturales pueden hincharse, pudrirse o debilitarse cuando están mojadas.
- **Estiramiento**: El cordel fino puede estirarse bajo una carga constante.

### Riesgos y seguridad

- **Enredos**: El cordel suelto puede enganchar herramientas o extremidades.
- **Cortes**: El cordel fino puede cortar la piel bajo tensión.

### Materiales relacionados

- **[Plant fibers](/es/materials/plant-fibers)**: Fuente bruta común.
- **[Animal hair](/es/materials/animal-hair)**: Fuente de fibra alternativa.
- **[Glue](/es/materials/glue)**: A menudo se utiliza con fijaciones.
