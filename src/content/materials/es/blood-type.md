El tipo de sangre es una clasificación biológica basada en antígenos específicos en los glóbulos rojos. Define qué sangre es compatible para transfusiones y cómo reacciona el sistema inmune ante sangre extraña.

## Descripción de cómo es

El tipo de sangre no es visible. Se define por los antígenos de superficie presentes en los glóbulos rojos, principalmente el sistema ABO (A, B, AB, O) y el Rh (positivo o negativo).

## Origen y dónde encontrarlo

- **Entornos**: En el torrente sanguíneo de humanos y muchos animales.
- **Señales**: Solo se identifica mediante pruebas de tipificación o registros genéticos.
- **Natural**: Se hereda y se mantiene estable toda la vida.

### Procesamiento mínimo requerido

- **Muestra**: Obtener una pequeña muestra de sangre con herramientas limpias.
- **Reactivos**: Mezclar con anticuerpos anti-A, anti-B y anti-D (Rh).
- **Observación**: Ver aglutinación para identificar el tipo.
- **Registro**: Etiquetar y documentar de inmediato para evitar errores.

### Herramientas necesarias para trabajarlo

- **Aguja o lanceta limpia**: Para una punción pequeña.
- **Láminas o tarjeta**: Para pruebas de aglutinación.
- **Solución salina**: Para dilución y limpieza.
- **Etiquetas y registros**: Para prevenir errores de identificación.
- **Almacenamiento fresco**: Conservación corta de muestras.

### Formas comunes de uso

- **Tipificación ABO/Rh**: Control estándar de compatibilidad.
- **Prueba cruzada**: Verificación previa a transfusión.
- **Identificación de emergencia**: Pulseras o registros para atención rápida.
- **Cuidado del embarazo**: Monitoreo Rh para prevenir sensibilización.

### Posibles sustitutos

- **Prueba cruzada directa**: Cuando no hay reactivos completos.
- **Compatibilidad por antecedentes**: Solo como medida temporal y de alto riesgo.

### Limitaciones y fallos comunes

- **Etiquetado incorrecto**: El error más peligroso en campo.
- **Reacciones débiles**: Reactivos pobres o bajo recuento celular dan falsos resultados.
- **Contaminación**: Herramientas sucias o muestras viejas distorsionan la prueba.
- **Tipificación incompleta**: Subtipos raros y otros antígenos pueden reaccionar.

### Riesgos y seguridad

- **Biohazard**: La sangre puede portar patógenos.
- **Lesión por aguja**: Riesgo de infección por pinchazos.
- **Reacción inmune**: Una transfusión incompatible puede ser fatal.
