Una teja (para techos) o baldosa (para suelos/paredes) es una pieza manufacturada de material resistente como cerámica, piedra, metal, arcilla cocida o incluso vidrio. Se utiliza generalmente para cubrir techos, suelos, paredes u otros objetos como mesas.

## Descripción de cómo es

Las baldosas suelen ser formas delgadas, cuadradas o rectangulares. Las **tejas** a menudo son curvas (como el estilo español o romano) para entrelazarse y canalizar el agua. Las **baldosas de suelo y pared** suelen ser planas, esmaltadas o sin esmaltar, y pueden ser muy decorativas. Son duras, frías al tacto y frágiles.

## Origen y dónde encontrarlo

Son productos manufacturados.

- **Tejas/Baldosas de Cerámica:** Hechas de **[Arcilla](/es/materials/clay)**, **[Arena](/es/materials/sand)** y otros minerales, luego cocidas en un **[Horno](/es/inventions/kiln)**.
- **Tejas/Baldosas de Hormigón:** Hechas de una mezcla de **[Cemento](/es/materials/cement)**, **[Arena](/es/materials/sand)** y **[Agua](/es/materials/water)**.
- **Baldosas de Piedra:** Cortadas de **[Piedra](/es/materials/stone)** natural (como pizarra, mármol, granito).

### Procesamiento mínimo requerido

1.  **Preparación:** Mezclar las materias primas (arcilla/agua o cemento/arena/agua) hasta obtener una consistencia plástica.
2.  **Moldeado:** Dar forma a la mezcla usando moldes o extrusión. Para las tejas, se forman curvas específicas.
3.  **Secado:** Dejar secar para eliminar la humedad.
4.  **Cocción/Curado:**
    - **Arcilla:** Cocer en un **[Horno](/es/inventions/kiln)** a altas temperaturas (vitrificación).
    - **Hormigón:** Dejar curar y endurecer con el tiempo.
5.  **Esmaltado (Opcional):** Aplicar una capa vidriada para impermeabilización y decoración antes de una segunda cocción.

### Herramientas necesarias para trabajar con ello

- **Moldes:** Marcos de madera o metal para dar forma.
- **Horno:** Esencial para tejas cerámicas.
- **Paleta:** Para aplicar **[Mortero](/es/materials/mortar)** para fijar las baldosas.
- **Cortador de baldosas:** Para cortar piezas y ajustarlas a espacios específicos.

### Formas comunes de uso

- **Techado:** Tejas superpuestas para repeler el agua. Formas comunes incluyen plana, pantile, romana y árabe (curva).
- **Pavimento:** Superficie duradera y fácil de limpiar para interiores y exteriores.
- **Paredes:** Revestimiento protector y decorativo para cocinas, baños y fachadas.
- **Drenaje:** Tejas porosas usadas para tuberías de drenaje.

### Posibles sustitutos

- **Tejas de [Madera](/es/materials/wood):** Más ligeras pero menos resistentes al fuego.
- **[Piedra](/es/materials/stone) (Pizarra):** Alternativa natural muy duradera para techos.
- **Láminas de [Metal](/es/materials/metal):** Techado ligero y duradero.
- **[Paja](/es/materials/straw):** Tallos de plantas usados para techado (aislante pero inflamable).
- **[Ladrillo](/es/materials/brick):** Puede usarse para pavimentos pero es más grueso.

### Limitaciones y fallos comunes

- **Fragilidad:** Se rompen fácilmente bajo impacto.
- **Peso:** Las tejas son pesadas y requieren una estructura de techo fuerte.
- **Daño por heladas:** Las tejas porosas pueden absorber agua y agrietarse en condiciones de congelación.
- **Resbaladizo:** Las baldosas esmaltadas pueden ser muy resbaladizas cuando están mojadas.

### Riesgos y seguridad

- **Objetos cayendo:** Las tejas sueltas pueden ser peligrosas.
- **Cortes:** Las baldosas rotas tienen bordes afilados.
- **Polvo de sílice:** Cortar baldosas genera polvo nocivo.

### Inventos relacionados

- **[Techado](/es/inventions/roofing):** El sistema principal donde se usan tejas.

- **[Mosaico](/es/inventions/mosaic):** Forma de arte usando pequeñas baldosas.
