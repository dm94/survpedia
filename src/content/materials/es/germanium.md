# Breve descripción

El germanio es un metaloide brillante, duro y frágil, de color blanco grisáceo. Fue el primer material utilizado para transistores antes de que el silicio se volviera dominante.

## Propiedades

-   **Semiconductividad:** Al igual que el silicio, su conductividad puede ser controlada.
-   **Óptica:** Transparente a la radiación infrarroja, lo que lo hace útil para cámaras térmicas.
-   **Refracción:** Alto índice de refracción.

## Uso

-   **Electrónica:** Usado en la primera generación de transistores y diodos (los diodos de germanio todavía se usan en radios de galena).
-   **Óptica:** Lentes para cámaras infrarrojas y espectroscopios.
-   **Comunicaciones:** Dopante en cables de fibra óptica.

## Fabricación/Extracción

Se extrae principalmente como subproducto del procesamiento de minerales de zinc.
1.  **Concentración:** Los concentrados de germanio se cloran para producir tetracloruro de germanio.
2.  **Purificación:** Se destila para obtener alta pureza.
3.  **Hidrólisis:** Se hidroliza a óxido (GeO2).
4.  **Reducción:** Se reduce con hidrógeno para producir germanio metálico.
