Una longitud larga y continua de fibras entrelazadas, utilizada para la producción de textiles, costura, ganchillo, tejido, bordado y fabricación de cuerdas.

## Descripción de cómo es

El hilo aparece como una hebra delgada y continua de fibras retorcidas. Puede ser suave o velloso, grueso o delgado, dependiendo de la fibra y el método de hilado. Tiene resistencia a la tracción pero es flexible.

## Origen y dónde encontrarlo

- **Entorno**: Creado en asentamientos o por viajeros con herramientas de hilado.
- **Señales**: Husos, ruecas o ovillos de hilo.
- **Transformación**: Requiere hilar fibras crudas.

### Procesamiento mínimo requerido

1. **Cardado/Peinado**: Alinear las fibras crudas (lana, algodón, lino).
2. **Hilado**: Torcer las fibras alineadas para unirlas y crear resistencia.
3. **Retorcido** (Opcional): Torcer múltiples hebras juntas para mayor resistencia.

### Herramientas necesarias para trabajarlo

- **Huso**: Un palo simple con un peso (tortera) para torcer fibras.
- **Rueca**: Una máquina para acelerar el proceso de hilado.
- **Cardas/Peines**: Para preparar las fibras.

### Formas comunes de uso

- **Cabo simple**: Para tejidos finos.
- **Multicabo**: Para tejer, telas más fuertes o hilo de coser.
- **Encerado**: Para coser cuero o impermeabilizar (ej. hilo de zapatero).

### Posibles sustitutos

- **Tendón**: Tendón animal (muy fuerte, bueno para coser).
- **Tripa**: Intestinos (para suturas o cuerdas).
- **Tiras de tela/cuero**: Para tejidos toscos o ataduras.
- **Mecha sin hilar**: Débil, se rompe fácilmente, pero puede usarse para tramas voluminosas.

### Limitaciones y fallos comunes

- **Destorcido**: Si no se fija correctamente, puede deshacerse.
- **Rotura**: Bajo demasiada tensión o si se hila de manera desigual.
- **Podredumbre**: El hilo de fibra natural se descompone con la humedad.

### Riesgos y seguridad

- **Enredos**: Puede convertirse en un lío si no se enrolla correctamente.
- **Cortes**: El hilo fino y fuerte puede cortar la piel bajo tensión.

### Materiales relacionados

- **Fibras vegetales**: Algodón, Lino, Cáñamo (materias primas).
- **Lana**: Pelo animal (materia prima).
- **Tela**: El producto de tejer hilo.
