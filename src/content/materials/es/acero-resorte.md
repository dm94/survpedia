# Descripción

Un tipo de acero de alto carbono o aleado con un límite elástico muy alto. Permite que los objetos hechos de acero de resorte vuelvan a su forma original a pesar de una desviación o torsión significativa.

## Descripción de cómo es

Parece acero normal pero es mucho más difícil de doblar permanentemente. Si se dobla dentro de su límite elástico, vuelve a su posición instantáneamente.

## Origen y dónde encontrarlo

- **Manufacturado:** No ocurre naturalmente. Debe ser forjado a partir de hierro y carbono con tratamiento térmico específico.
- **Recolección:** Resortes de suspensión de autos viejos, hojas de sierra, cuerdas de piano.

### Procesamiento mínimo requerido

- **Templado:** Calentar a temperatura crítica y enfriar bruscamente.
- **Revenido:** Recalentar a una temperatura más baja para reducir la fragilidad y aumentar la tenacidad.

### Herramientas necesarias para trabajarlo

- **Fragua:** Para el tratamiento térmico.
- **Amoladora:** Para dar forma (difícil de limar).
- **Aceite/Agua:** Para el enfriamiento.

### Formas comunes de uso

- **Muelles helicoidales:** Para suspensión o mecanismos.
- **Muelles planos:** Para ballestas o muelles reales de relojería.

### Posibles sustitutos

- **Bronce endurecido:** Para muelles no magnéticos.
- **Madera/Bambú:** Para arcos o trampas primitivas (menor durabilidad).

### Limitaciones y fallos comunes

- **Fatiga:** El ciclo repetido puede causar grietas y fallos.
- **Óxido:** El alto contenido de carbono lo hace propenso a la corrosión si no se engrasa o pinta.
- **Sobrecalentamiento:** Pérdida del temple si se calienta demasiado al amolar.

### Riesgos y seguridad

- **Rotura:** Puede romperse violentamente bajo tensión, liberando energía almacenada.
- **Fragmentos:** Fragmentos duros pueden volar si se rompe.
