# Breve descripción

El papel de lija es un material abrasivo recubierto que consiste en hojas de papel o tela con material abrasivo pegado a una cara. Se utiliza para eliminar pequeñas cantidades de material de las superficies, ya sea para hacerlas más lisas (por ejemplo, en pintura y acabado de madera), para eliminar una capa de material (como pintura vieja), o a veces para hacer la superficie más rugosa (por ejemplo, como preparación para el encolado).

## Descripción de cómo es

Generalmente es una hoja de papel grueso de color marrón o amarillento con una superficie rugosa y arenosa en un lado. La rugosidad varía desde muy gruesa (como grava pequeña) hasta muy fina (como polvo).

## Origen y dónde encontrarlo

- **Entornos:** Es un producto manufacturado, pero sus componentes se pueden encontrar en la naturaleza.
- **Componentes:** Arena (sílice), vidrio molido, granate o esmeril; papel o cuero; pegamento o resina.
- **Es un material compuesto.**

### Procesamiento mínimo requerido

1.  **Preparación del soporte:** Obtener papel resistente, tela o cuero curtido.
2.  **Preparación del abrasivo:** Cribar arena o triturar vidrio/piedra hasta obtener un tamaño de grano uniforme.
3.  **Aplicación:** Aplicar una capa de pegamento al soporte y espolvorear el abrasivo uniformemente antes de que el pegamento se seque.
4.  **Curado:** Dejar secar completamente.

### Herramientas necesarias para trabajarla

- **Cribas/Tamices:** Para clasificar el tamaño del grano abrasivo (el "grano" de la lija).
- **Brocha:** Para aplicar el pegamento.
- **Tijeras/Cuchillo:** Para cortar el papel de lija al tamaño deseado.

### Formas comunes de uso

- **Hojas:** Para uso manual o en bloques de lijado.
- **Cintas/Discos:** Para máquinas lijadoras (versiones modernas).
- **Esponjas:** Bloques flexibles recubiertos de abrasivo.

### Posibles sustitutos

- **Piel de tiburón (Zapa):** Usada históricamente como lija natural muy efectiva.
- **Cola de caballo (Equisetum):** Una planta con alto contenido de sílice usada para pulir.
- **Arena y un trapo:** Frotar arena suelta con un paño o cuero.
- **Piedra pómez:** Para un alisado fino.

### Limitaciones y fallos comunes

- **Embotamiento:** El polvo del material lijado puede obstruir los espacios entre los granos abrasivos, haciendo que la lija pierda eficacia.
- **Desgaste:** Los granos abrasivos se desprenden con el uso.
- **Humedad:** El papel normal se deshace si se moja (se requiere papel impermeable para lijado en húmedo).

### Riesgos y seguridad

- **Polvo:** El lijado produce polvo fino que puede ser perjudicial si se inhala (usar máscara).
- **Abrasión de la piel:** Puede raspar la piel si se maneja sin cuidado.

### Materiales relacionados

- **Arena:** El abrasivo más básico.
- **Vidrio:** Vidrio molido usado para "papel de vidrio".
- **Pegamento:** Necesario para fijar el abrasivo.
- **Papel/Tela:** El material de soporte.
