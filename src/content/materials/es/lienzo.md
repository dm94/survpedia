El lienzo es una tela de tejido liso extremadamente duradera que se utiliza para fabricar velas, tiendas de campaña, marquesinas, mochilas y otros artículos para los que se requiere robustez, así como en objetos de moda como bolsos, fundas para dispositivos electrónicos y zapatos. También es popularmente utilizado por artistas como superficie de pintura, típicamente estirada sobre un marco de madera.

## Descripción física

El lienzo es una tela pesada y gruesa. A diferencia de otras telas de algodón pesadas como la mezclilla, el lienzo tiene un tejido liso en lugar de un tejido de sarga. Viene en dos tipos básicos: liso y "duck" (del holandés 'doek', que significa tela), siendo el "duck" un tejido más apretado.

## Origen y dónde encontrarlo

- **Fibras**: Originalmente hecho de cáñamo (la palabra lienzo/canvas proviene de la palabra árabe para cannabis). El lienzo moderno suele estar hecho de algodón o lino.
- **Fuente**: Tallos de plantas (cáñamo, lino) o cápsulas (algodón).

### Procesamiento mínimo requerido

1.  **Cosecha**: Recolección de las fibras vegetales.
2.  **Hilado**: Torcer las fibras en hilos fuertes y gruesos.
3.  **Tejido**: Usar un telar para entrelazar los hilos en un patrón simple de arriba a abajo (tejido liso).
4.  **Imprimación (para Arte)**: Aplicar una cola (pegamento) y una base (gesso/imprimación) para evitar que la pintura al óleo pudra la tela.

### Herramientas necesarias para trabajarlo

- **Telar**: Para tejer la tela.
- **Aguja e Hilo**: Agujas resistentes (agujas de velero) para coser.
- **Bastidores**: Marcos de madera para tensar el lienzo para pintar.

### Formas comunes de uso

- **Lienzo crudo**: Color beige/blanquecino, usado para bolsas o ropa de trabajo.
- **Lienzo imprimado**: Superficie blanca y rígida lista para pintar.
- **Lienzo encerado**: Tratado con cera para resistencia al agua.

### Posibles sustitutos

- **Mezclilla (Denim)**: Otra tela de algodón fuerte, pero con una costilla diagonal (sarga).
- **Cuero**: Más fuerte pero más pesado y difícil de coser.
- **Paneles de madera**: Alternativa rígida para pintar.

### Limitaciones y fallos comunes

- **Podredumbre**: Las fibras naturales se pudrirán si se dejan húmedas (moho).
- **Desgarro**: Aunque es fuerte, puede rasgarse bajo fuerza aguda.
- **Aflojamiento**: Los cambios de humedad pueden hacer que el lienzo estirado se afloje o se tense demasiado.

### Riesgos y seguridad

- **Inflamabilidad**: El algodón/lino sin tratar arde fácilmente.
- **Polvo**: Cortar o rasgar lienzo seco produce pelusa.

### Materiales relacionados

- **Algodón/Lino/Cáñamo**: Las materias primas.
- **Pintura**: A menudo aplicada al lienzo.
- **Madera**: Usada para bastidores.
- **Cera**: Usada para impermeabilización.
