La cuerda es un cordón delgado hecho al retorcer fibras en una hebra continua. Se utiliza para atar, sujetar y unir materiales cuando se necesita un conector flexible.

## Descripción física

La cuerda es ligera y flexible. Su resistencia depende del tipo de fibra y del número de hebras trenzadas. Permite nudos, envolturas y tensado sin romperse si está bien hecha.

## Origen y dónde encontrarlo

- **Entornos**: Donde existan fuentes de fibras (plantas o animales).
- **Señales**: Hebras retorcidas o trenzadas hechas con fibras o tendón.
- **Es un material procesado.**

### Procesamiento mínimo requerido

1.  **Recolección de fibras**: Fibras vegetales, pelo animal o tendón.
2.  **Preparación**: Limpieza y alineado de fibras.
3.  **Retorcido**: Girar fibras para formar una hebra.
4.  **Trenzado**: Unir varias hebras para mayor resistencia.

### Herramientas necesarias para trabajarla

- Manos o husos simples para retorcer.
- Ganchos o estacas para mantener tensión.
- Cuchillos para cortar fibras.

### Formas comunes de uso

- **Hebra simple**: Ataduras ligeras y temporales.
- **Dos o tres hebras**: Cordón más resistente para herramientas y costura.
- **Trenzada**: Cinta plana o redonda para mayor durabilidad.

### Posibles sustitutos

- **[Hilo](/es/materials/yarn)**: Cordón más grueso y suave.
- **[Fibras vegetales](/es/materials/plant-fibers)**: Usadas directamente como cordaje.
- **[Cuero](/es/materials/leather)**: Tiras de cuero crudo para amarres fuertes.

### Limitaciones y fallos comunes

- **Abrasión**: Las fibras se deshilachan con el roce.
- **Humedad**: Las fibras naturales se hinchan, pudren o debilitan.
- **Estiramiento**: La cuerda fina se alarga con carga constante.

### Riesgos y seguridad

- **Enredos**: La cuerda suelta puede enganchar herramientas o extremidades.
- **Cortes**: Los cordones finos pueden cortar la piel bajo tensión.

### Materiales relacionados

- **[Fibras vegetales](/es/materials/plant-fibers)**: Fuente común de fibras.
- **[Pelo animal](/es/materials/pelo-animal)**: Fuente alternativa de fibras.
- **[Pegamento](/es/materials/glue)**: A menudo se usa con amarres.
