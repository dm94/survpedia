El aceite vegetal es un triglicérido extraído de plantas, comúnmente de semillas, nueces o frutas. Es un material fundamental para la nutrición, la iluminación y la ingeniería química primitiva.

## Descripción de cómo es

Un líquido viscoso a temperatura ambiente, que generalmente varía en color de transparente a amarillo o verde. Se siente grasiento y resbaladizo. A diferencia de la grasa animal, generalmente permanece líquido en temperaturas más frías.

## Origen y dónde encontrarlo

- **Entornos**: Tierras agrícolas, arboledas silvestres (olivos, nueces).
- **Fuentes**: Aceitunas, semillas de girasol, colza, semillas de lino, nueces, cocos.
- **Natural**: Contenido dentro de las semillas/frutas pero requiere extracción.

### Procesamiento mínimo necesario

- **Cosecha**: Recolección de las semillas o frutas ricas en aceite.
- **Molienda**: Trituración del material hasta formar una pasta usando un **[Molino](/es/inventions/mill)** o **[Molino de Mano](/es/inventions/hand-mill)**.
- **Prensado**: Aplicación de alta presión a la pasta para exprimir el aceite.
- **Decantación**: Dejar reposar el aceite para que las impurezas se asienten en el fondo.

### Herramientas necesarias para trabajarla

- **[Molino de Mano](/es/inventions/hand-mill)** o **[Mortero](/es/inventions/mortar)**: Para triturar.
- **[Prensa Mecánica](/es/inventions/mechanical-press)**: Esencial para una extracción eficiente.
- **[Contenedores](/es/inventions/containers)**: Para recolección y almacenamiento.
- **Tela**: Para filtrar.

### Formas comunes de uso

- **Aceite Comestible**: Para freír y aderezar alimentos.
- **Aceite para Lámparas**: Combustible para **[Lámparas de Aceite](/es/inventions/oil-lamp)** simples.
- **Aceite Infusionado**: Base para extractos de **[Plantas Medicinales](/es/knowledge/medicinal-plants)**.
- **Tratamiento de Madera**: Recubrimiento protector para madera.

### Posibles sustitutos

- **[Grasa Animal](/es/materials/animal-fat)**: Se puede usar para cocinar, combustible y jabón, pero es sólida a temperatura ambiente.
- **Productos derivados del petróleo**: Para lubricación y combustible (moderno).

### Limitaciones y fallos comunes

- **Rancidez**: Se oxida y se estropea si se expone a la luz, el calor o el aire durante demasiado tiempo.
- **Eficiencia de Extracción**: Sin una buena prensa, obtener aceite es muy laborioso y derrochador.
- **Punto de Humo**: Diferentes aceites se queman a diferentes temperaturas; algunos humean fácilmente cuando se calientan.

### Riesgos y seguridad

- **Fuego**: Altamente inflamable, especialmente cuando se calienta. Los incendios de aceite no se pueden apagar con agua.
- **Deterioro**: Consumir aceite rancio no es saludable.

### Materiales relacionados

- **[Grasa Animal](/es/materials/animal-fat)**: La contraparte animal.
- **[Jabón](/es/materials/soap)**: Hecho de aceite y álcali.
- **[Plantas Medicinales](/es/knowledge/medicinal-plants)**: A menudo infusionadas en aceite.
