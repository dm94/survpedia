# Descripción

Un tipo de acero de alto carbono o aleado con un límite elástico muy alto. Permite que los objetos hechos de acero para resortes vuelvan a su forma original a pesar de una deflexión o torsión significativa.

## Descripción de cómo es

Se parece al acero normal pero es mucho más difícil de doblar permanentemente. Si se dobla dentro de su límite elástico, vuelve a su posición instantáneamente.

## Origen y dónde encontrarlo

- **Fabricado:** No ocurre naturalmente. Debe ser forjado a partir de hierro y carbono con un tratamiento térmico específico.
- **Recolección:** Resortes de suspensión de coches viejos, hojas de sierra, cuerda de piano.

### Procesamiento mínimo requerido

- **Endurecimiento:** Calentar a temperatura crítica y templar.
- **Revenido:** Recalentar a una temperatura más baja para reducir la fragilidad y aumentar la tenacidad.

### Herramientas necesarias para trabajarlo

- **Fragua:** Para tratar térmicamente.
- **Amoladora:** Para dar forma (difícil de limar).
- **Aceite/Agua:** Para el templado.

### Formas comunes de uso

- **Resortes helicoidales:** Para suspensión o mecanismos.
- **Resortes planos:** Para ballestas o muelles reales de relojería.

### Sustitutos posibles

- **Bronce endurecido por trabajo:** Para resortes no magnéticos.
- **Madera/Bambú:** Para arcos o trampas primitivas (menor durabilidad).

### Limitaciones y fallos comunes

- **Fatiga:** El ciclo repetido puede causar grietas y fallos.
- **Óxido:** El alto contenido de carbono lo hace propenso a la corrosión si no se engrasa o pinta.
- **Sobrecalentamiento:** Pierde el temple si se calienta demasiado durante el esmerilado.

### Riesgos y seguridad

- **Rotura violenta:** Puede romperse violentamente bajo tensión, liberando energía almacenada.
- **Fragmentos:** Los fragmentos duros pueden volar si se rompe.
