El cuerno es una excrecencia dura y permanente que se encuentra en la cabeza del ganado vacuno, ovejas, cabras y búfalos. Consiste en una vaina de queratina (la misma proteína que el pelo y las uñas) que cubre un núcleo óseo vivo. Fue uno de los primeros termoplásticos de la humanidad.

## Descripción de cómo es

- **Apariencia:** Varía de negro, gris y marrón a amarillo pálido translúcido o blanco. A menudo tiene grano o capas.
- **Textura:** Suave cuando se pule, fibrosa si se rompe.
- **Propiedades:** Es duro y resistente. Únicamente, cuando se calienta (hervido o asado), se vuelve blando y flexible, lo que permite aplanarlo, moldearlo o soldarlo. Cuando se enfría, conserva la nueva forma.

## Origen y dónde encontrarlo

- **Fuente:** Ganado doméstico, bueyes, ovejas, cabras y búfalos de agua.
- **Distinción:** A diferencia del **[Asta de ciervo](/es/materials/antler)** (que es hueso sólido y se cae anualmente), el cuerno es hueco (una vez que se retira el núcleo) y permanente.

### Procesamiento mínimo requerido

1.  **Extracción:** Se retira el cuerno del animal (generalmente después del sacrificio).
2.  **Maceración:** Se deja pudrir o se hierve para que el núcleo óseo interno se separe de la vaina de queratina.
3.  **Limpieza:** Se raspa para eliminar la suciedad y las capas externas rugosas.
4.  **Calentamiento:** Se hierve en agua o se calienta sobre fuego para ablandar la queratina.
5.  **Modelado:**
    - **Aplanado:** Se abre con un corte y se prensa entre placas calientes para hacer láminas planas (para peines, paneles de linternas).
    - **Moldeado:** Se prensa en moldes para cucharas o tazas.
    - **Soldadura:** Dos piezas se pueden fusionar bajo calor y presión.

### Herramientas necesarias para trabajarla

- **Sierra:** Para cortar el cuerno a la longitud deseada.
- **Caldero:** Para hervir agua/aceite y ablandarlo.
- **Tornillo de banco/Prensa:** Para aplanar o moldear el cuerno caliente.
- **Raspadores/Cuchillos:** Para alisar la superficie.
- **Abrasivos:** **[Arena](/es/materials/arena)** o **[Piedra pómez](/es/materials/pumice)** para pulir.

### Formas comunes de uso

- **Sólido:** Las puntas son sólidas y se usan para mangos o botones.
- **Hueco:** El cuerpo principal se usa para tazas (cuernos para beber) o cuernos de pólvora.
- **Lámina:** El cuerno aplanado se usa para **[Peines](/es/inventions/comb)**, marcos de gafas y, antes del vidrio, como ventanas translúcidas para linternas.
- **Compuesto:** Se usan capas de cuerno en el vientre de los arcos compuestos para almacenar energía.

### Posibles sustitutos

- **[Plástico](/es/materials/plastic):** El equivalente sintético moderno.
- **[Hueso](/es/materials/bone):** Más duro y quebradizo, no se puede moldear con calor.
- **[Madera](/es/materials/wood):** Más fácil de trabajar pero carece de las propiedades termoplásticas y la translucidez.
- **Carey:** Un material de queratina similar, pero de tortugas (a menudo ilegal/poco ético ahora).

### Limitaciones y fallos comunes

- **Delaminación:** El cuerno viejo o seco puede pelarse en capas.
- **Insectos:** La queratina es alimento para las polillas de la ropa y los escarabajos de las alfombras.
- **Calor:** Se deformará si se expone al calor nuevamente. Huele a pelo quemado cuando se quema.

### Riesgos y seguridad

- **Quemaduras:** Trabajar con agua hirviendo y cuerno caliente.
- **Polvo:** Respirar polvo de queratina durante el lijado puede ser irritante.

### Materiales relacionados

- **[Hueso](/es/materials/bone):** A menudo se usa junto con él.
- **[Pegamento](/es/materials/glue):** Las virutas de cuerno se pueden hervir para hacer pegamento animal fuerte.
- **Queratina:** La proteína base.
