# Breve descripción

Un instrumento de dibujo técnico que se utiliza para trazar círculos o arcos. Como divisores, también se utilizan como herramienta para medir distancias, en particular en mapas. Consta de dos partes (patas) conectadas por una bisagra que se pueden ajustar para cambiar el radio del círculo dibujado o la distancia medida.

## Uso / Función

-   **Geometría**: Dibujar círculos y arcos perfectos.
-   **Navegación**: Medir distancias en mapas (usando divisores).
-   **Carpintería**: Marcar medidas y trazar (transferir perfiles).
-   **Albañilería**: Diseñar patrones de piedra.

## Principio de funcionamiento

El compás funciona según el principio de un radio fijo. Al fijar una pata (la punta de aguja) y rotar la otra pata (la punta de dibujo) a su alrededor, se crea un círculo. La distancia entre las dos patas determina el radio. La fricción en la bisagra mantiene las patas en un ángulo constante.

## Cómo crearlo

1.  **Patas**: Da forma a dos patas rectas de madera, hueso o metal. Deben ser robustas y rectas.
2.  **Bisagra**: Conecta las dos patas en la parte superior.
    -   *Simple*: Átalas juntas firmemente con cuerda/cuero.
    -   *Mejor*: Usa un remache o un tornillo que permita el movimiento pero proporcione suficiente fricción para mantener la posición.
3.  **Puntas**:
    -   *Punta de Anclaje*: Afila una pata en punta o coloca una aguja de metal.
    -   *Punta de Dibujo**: La otra pata también puede ser afilada (para raspar/trazar) o tener un soporte para carbón, plomo o tinta.

## Materiales necesarios

-   **Patas**: Madera dura, Hueso, Latón, Hierro o Acero.
-   **Bisagra**: Tira de cuero, remache o tornillo.
-   **Puntas**: Acero endurecido o hierro para las puntas; carbón o plomo para dibujar.

## Variantes y mejoras

-   **Divisores**: Ambas patas tienen puntas afiladas. Se utiliza únicamente para medir y transferir distancias.
-   **Compás de Vara**: Dos puntos se deslizan a lo largo de una vara (barra) para dibujar círculos muy grandes.
-   **Compás de Muelle**: Un arco de resorte mantiene las patas separadas y un tornillo ajusta la distancia, ofreciendo mayor precisión.
-   **Calibradores**: Patas curvas para medir diámetros exteriores o interiores de objetos.

## Límites y riesgos

-   **Deslizamiento**: La punta de anclaje puede resbalar si no se presiona firmemente, arruinando el dibujo.
-   **Precisión**: Una bisagra floja conduce a círculos inexactos.
-   **Tamaño**: Limitado por la longitud de las patas (a menos que se use un compás de vara).
