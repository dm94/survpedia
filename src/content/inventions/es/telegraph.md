# Descripción breve

El telégrafo es un sistema de comunicación que transmite señales eléctricas a través de cables de un lugar a otro. Traduce una acción mecánica (golpear una llave o manipulador) en pulsos eléctricos que producen un sonido o una marca en el extremo receptor.

## Uso / Función

-   **Uso principal**: Comunicación instantánea a larga distancia.
-   **Usos secundarios**: Coordinación de transporte (trenes), mando militar, sincronización horaria.
-   **Escala**: Desde local (habitación a habitación) hasta continental.

## Principio de funcionamiento

**Electromagnetismo**:
1.  **El Circuito**: Una batería se conecta a un bucle de cable que se extiende entre dos estaciones.
2.  **El Transmisor (Manipulador)**: Un interruptor que normalmente rompe el circuito. Cuando se presiona, completa el circuito, permitiendo que fluya la electricidad.
3.  **El Receptor (Sonda)**: Un electroimán (una bobina de alambre envuelta alrededor de una pieza de hierro). Cuando la electricidad fluye a través de la bobina, esta se vuelve magnética y atrae un brazo de hierro móvil (armadura).
4.  **La Señal**: La armadura golpea un tope, haciendo un sonido de "clic". Cuando se suelta la llave, un resorte tira de la armadura hacia atrás.
5.  **El Código**: Patrones de clics cortos y largos (puntos y rayas) representan letras y números (por ejemplo, el Código Morse).

## Cómo crearlo

### 1. El Electroimán (Receptor)
-   Envuelve alambre de cobre aislado alrededor de un núcleo de hierro blando (como un clavo grande o un perno).
-   Deja dos extremos del alambre expuestos para conectar al circuito.
-   Móntalo sobre una base de madera.

### 2. La Sonda (Sounder)
-   Monta una tira de metal flexible (hierro o acero) o un brazo de metal con bisagras sobre el electroimán.
-   El metal debe estar lo suficientemente cerca como para ser atraído cuando el imán está encendido, pero volver a subir por un resorte cuando está apagado.
-   Añade un "tope" (un tornillo o clavo) para que el metal golpee y haga un sonido nítido.

### 3. El Manipulador (Transmisor)
-   Crea un interruptor de resorte simple usando una tira de metal atornillada a un bloque de madera.
-   Cuando se empuja hacia abajo, debe tocar un punto de contacto (otro tornillo) para cerrar el circuito.

### 4. Conexión
-   Conecta la Batería, el Manipulador y el Electroimán en un solo circuito en serie.
-   Para largas distancias, tiende dos cables entre las estaciones, o usa un cable y la Tierra (suelo) como camino de retorno.

## Materiales necesarios

-   **Alambre de Cobre Aislado**: Para las bobinas y líneas de transmisión.
-   **Hierro Blando**: Para el núcleo del electroimán (clavos, pernos). El hierro blando pierde magnetismo rápidamente cuando se corta la energía (esencial para una señalización rápida).
-   **Tira de Metal Ferroso**: Para la armadura (puede ser de una lata o chatarra de hierro).
-   **Madera**: Para la base y la estructura.
-   **Fuente de Energía**: Una batería (pila voltaica o celda Daniell).

## Variantes y mejoras

-   **Retorno por Tierra**: Usar el suelo en lugar de un segundo cable ahorra el 50% del cobre.
-   **Relé**: Para distancias muy largas, la señal se debilita. Un relé es un receptor telegráfico sensible que actúa como interruptor para un circuito de batería local nuevo, repitiendo la señal con plena potencia a la siguiente estación.
-   **Sonda vs. Escritor**: Las primeras versiones marcaban cinta de papel; los operadores posteriores preferían escuchar los clics ("Sonda").

## Límites y riesgos

-   **Resistencia**: Los cables finos a largas distancias tienen alta resistencia, debilitando la corriente. Necesitas cables más gruesos, mayor voltaje o relés.
-   **Privacidad**: Cualquiera que se conecte al cable puede leer el mensaje.
-   **Habilidad**: Requiere aprender un código (como Morse) para usarlo eficazmente.
-   **Clima**: Los rayos que golpean la línea pueden destruir el equipo y matar a los operadores. Los pararrayos (espacios de chispa) son esenciales.
