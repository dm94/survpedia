# Descripción breve

El ábaco es una herramienta de cálculo manual que utiliza cuentas o piedras desplazadas a lo largo de varillas o ranuras para realizar operaciones aritméticas. Es uno de los dispositivos de computación más antiguos conocidos, anterior al sistema escrito de numeración arábiga.

## Uso / Función

- **Cálculo**: Realizar sumas, restas, multiplicaciones y divisiones.
- **Educación**: Enseñar los conceptos de valor posicional y aritmética a los estudiantes.
- **Comercio**: Calcular rápidamente precios y totales en mercados y tiendas.
- **Escala**: Uso individual y doméstico.

## Principio de funcionamiento

El ábaco explota el concepto de **valor posicional**. Cada varilla o alambre representa una potencia diferente de diez (unidades, decenas, centenas, millares, etc.). Al mover las cuentas para representar números y seguir algoritmos específicos para desplazarlas durante las operaciones, se pueden realizar cálculos complejos rápidamente.

## Cómo crearlo

- **Versión mínima funcional**: Una serie de ranuras en la arena con pequeñas piedras (calculi) movidas dentro de ellas.
- **Esencial**: Un marco para mantener unidos los componentes y un conjunto de contadores móviles.
- **Nivel técnico**: Básico.

## Materiales necesarios

- **Materiales esenciales**:
  - **[Madera](/es/materials/wood)**: Para el marco y posiblemente las varillas.
  - **[Piedra](/es/materials/stone)** o **[Arcilla](/es/materials/clay)**: Para las cuentas o contadores.
- **Herramientas**:
  - **[Cuchillo](/es/inventions/knife)** o **[Sierra](/es/inventions/saw)**: Para dar forma a la madera.
  - **[Taladro](/es/inventions/hand-drill)**: Para hacer agujeros en las cuentas.
- **Posibles sustitutos**:
  - **[Alambre](/es/materials/wire)**: En lugar de varillas de madera para un movimiento más suave.
  - **[Cuerda](/es/materials/string)**: Para sostener las cuentas si no se dispone de una varilla rígida.

## Variantes y mejoras

- **Ábaco Romano**: Utilizaba ranuras en una placa de metal con contadores deslizantes.
- **Suanpan (Chino)**: Generalmente tiene dos cuentas en la cubierta superior y cinco en la inferior (sistema 2:5), lo que permite cálculos hexadecimales.
- **Soroban (Japonés)**: Versión mejorada con una configuración de cuentas 1:4, optimizada para el sistema decimal.
- **Schoty (Ruso)**: Utiliza varillas horizontales y un sistema de 10 cuentas.

## Límites y riesgos

- **Entrenamiento**: Requiere una práctica significativa para alcanzar altas velocidades y precisión.
- **Error Manual**: Un desliz del dedo puede cambiar el resultado de un cálculo largo.
- **Sin Memoria**: A diferencia de las computadoras modernas, no almacena resultados o pasos intermedios a menos que se registren en otro lugar.
