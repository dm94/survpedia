# Breve descripción

Una honda es un arma de proyectiles utilizada típicamente para lanzar un proyectil contundente como una piedra, arcilla o "bala de honda" de plomo. Es conocida por su simplicidad, alcance y poder letal (famosamente usada por David contra Goliat).

## Uso / Función

- **Combate**: Arma efectiva de largo alcance capaz de causar traumatismos graves.
- **Caza**: Se usa para cazar animales pequeños o proteger rebaños de depredadores.

## Principio de funcionamiento

La honda funciona extendiendo la longitud del brazo humano, permitiendo que la piedra se acelere a una velocidad mucho mayor de la que se podría lograr lanzándola solo con la mano. El hondero hace girar la honda para aumentar la velocidad y suelta una cuerda en el momento preciso para lanzar el proyectil tangencialmente.

## Cómo crearlo

1.  **Cuerdas**: Trenza dos longitudes de cuerda fuerte de fibras vegetales (cáñamo, lino) o corta tiras de cuero. Una cuerda debe tener un bucle para un dedo, la otra un nudo o pestaña para agarrar.
2.  **Bolsa**: Crea una bolsa en el centro para sostener la piedra. Esto puede ser una pieza más ancha de cuero o una sección dividida/tejida de las cuerdas.
3.  **Ensamblaje**: Cose o ata las cuerdas de forma segura a la bolsa. La longitud total suele ser desde el pecho hasta el suelo o la longitud del brazo.

## Materiales necesarios

- **Cordaje**: Fibras vegetales, tiras de cuero o lana.
- **Material de la bolsa**: Cuero, tela gruesa o fibras tejidas.
- **Munición**: Piedras redondas y lisas (rocas de río) o balas de arcilla/plomo moldeadas.

## Variantes y mejoras

- **Fustíbalo (Staff Sling)**: Una honda unida al extremo de un bastón para un alcance y palanca aún mayores.
- **Balas de plomo**: Fundir proyectiles de plomo mejora significativamente el alcance y el impacto sobre las piedras debido a una mayor densidad.

## Límites y riesgos

- **Habilidad**: Requiere una práctica significativa para lograr precisión y el momento adecuado.
- **Seguridad**: Riesgo de golpearse a uno mismo o a aliados si se suelta mal.
- **Espacio**: Requiere espacio para balancear la honda.
