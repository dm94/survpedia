# Breve descripción

Un tornillo de banco (o prensa de banco) es un aparato mecánico utilizado para asegurar un objeto y permitir realizar trabajos sobre él. Los tornillos de banco tienen dos mordazas paralelas, una fija y la otra móvil, que se acercan o alejan mediante un tornillo y una palanca.

## Uso / Función

- **Sujeción:** Asegurar firmemente las piezas de trabajo para aserrar, limar, taladrar o pegar.
- **Trabajo de metales:** Sujetar metal para martillar o doblar.
- **Carpintería:** Sujetar madera para cepillar o aserrar.
- **Escala:** Esencial para cualquier taller.

## Principio de funcionamiento

Un tornillo grande (husillo) mueve una mordaza deslizante hacia una mordaza fija. La ventaja mecánica del tornillo permite al usuario aplicar una fuerza de sujeción masiva con un esfuerzo moderado en el mango.

## Cómo crearlo

### Versión primitiva (Tornillo de carpintero)

1.  **Mordazas:** Dos bloques gruesos de madera dura.
2.  **Tornillo:** Un gran tornillo de madera (puede tallarse o hacerse con un macho y una terraja).
3.  **Guía:** Dos varillas lisas (madera o metal) para mantener las mordazas paralelas.
4.  **Montaje:** La mordaza fija se une a la pata del banco de trabajo.

### Versión avanzada (Tornillo de banco de maquinista)

1.  **Cuerpo:** Cuerpo de hierro fundido (mordaza fija) atornillado al banco.
2.  **Corredera:** Una mordaza móvil sobre una guía cuadrada o de cola de milano.
3.  **Husillo:** Un tornillo de acero pesado con rosca trapezoidal (Acme).
4.  **Mordazas:** Placas frontales de acero endurecido con estriado cruzado para agarre (reemplazables).

## Materiales necesarios

- **Hierro fundido:** Para el cuerpo (masa y resistencia).
- **Acero:** Para el husillo y los insertos de las mordazas.
- **Madera:** Para tornillos de banco más suaves o almohadillas de mordaza (para proteger trabajos delicados).

## Variantes y mejoras

- **Tornillo de Banco:** El estándar para maquinistas.
- **Mordaza de Máquina:** De perfil bajo, rectificada con precisión, utilizada en **[Fresadoras](/es/inventions/milling-machine)** y **[Taladros de Columna](/es/inventions/drill-press)**.
- **Tornillo para Tubos:** Mordazas en forma de V para sujetar tubos redondos.
- **Tornillo de Pie:** Tiene una pata que llega al suelo para soportar la fuerza de martillazos pesados (herrería).

## Límites y riesgos

- **Apriete excesivo:** Puede aplastar la pieza de trabajo o estropear las roscas del tornillo.
- **Marcado:** Las mordazas de acero duro marcarán materiales blandos (aluminio, madera). Use "mordazas blandas" (almohadillas de cobre/madera).
- **Grietas:** Los tornillos de banco de hierro fundido pueden agrietarse si se golpean fuerte con un martillo (a menos que sea un tornillo de pie específico de herrero).
