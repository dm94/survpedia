# Breve descripción

Las trampas para topos son dispositivos mecánicos colocados en túneles activos para capturar o eliminar topos y reducir daños en cultivos, raíces y líneas de riego.

## Uso / Función

- **Control de plagas:** Detiene galerías que debilitan suelo y raíces.
- **Protección de cultivos:** Evita pérdidas de plántulas y raíces.
- **Protección de infraestructuras:** Reduce daños en caminos, riego y taludes.
- **Escala:** Huerto, jardín y finca.

## Principio de funcionamiento

La trampa se coloca dentro del túnel y se activa por movimiento o presión sobre una placa o alambre disparador. La energía almacenada (muelle, alambre tensado o peso) se libera y cierra un lazo o mandíbulas sobre el túnel.

## Cómo crearla

1. **Localiza un túnel activo:** Presiona la superficie y abre una sección pequeña.
2. **Elige un mecanismo:** Lazo de alambre, mandíbulas tipo tijera o una caída con peso.
3. **Prepara una guía:** Usa **[Alambre](/es/inventions/wire)** o guías de madera para centrar el disparador.
4. **Ajusta el disparo:** Disparador ligero y equilibrado que se active con poca presión.
5. **Estabiliza:** Compacta la tierra alrededor para evitar vibraciones y bloquear la luz.
6. **Revisa a diario:** Reajusta o retira cuando se active.

## Materiales necesarios

- **Esenciales:** Armazón o estacas de **[Madera](/es/materials/wood)**, **[Acero](/es/materials/steel)** o **[Hierro](/es/materials/iron)** para muelle o mandíbulas.
- **Herramientas:** Pala, cuchillo, martillo, alicates.
- **Sustitutos:** **[Fibras vegetales](/es/materials/plant-fibers)** o **[Cuerda](/es/inventions/rope)** para lazos, una **[Piedra](/es/materials/stone)** como peso.

## Variantes y mejoras

- **Trampa de tijera:** Mandíbulas con muelle para muerte rápida.
- **Trampa de lazo:** Lazo simple que se cierra al paso.
- **Trampa de caída:** Piedra o tronco pesado disparado por un gatillo.
- **Caja de túnel:** Caja cubierta que bloquea la luz y protege a mascotas.

## Límites y riesgos

- **Captura no deseada:** Puede afectar a mascotas o fauna útil si se coloca mal.
- **Seguridad:** Las trampas con muelle pueden lesionar las manos.
- **Condiciones del suelo:** Suelo suelto o húmedo desajusta el disparador.
- **Ética y legalidad:** Algunas zonas limitan las trampas letales.
