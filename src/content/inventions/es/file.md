# Breve descripción

Herramienta de metalurgia y carpintería utilizada para cortar cantidades finas de material de una pieza de trabajo. Consiste en una barra de acero endurecido con una serie de dientes o crestas afiladas y paralelas.

## Uso / Función

Su función principal es la abrasión y el alisado.

- **Afilado**: Esencial para mantener otras herramientas como sierras, hachas y cuchillos.
- **Alisado**: Eliminación de rebabas y bordes ásperos de objetos de metal o madera.
- **Dar forma**: Eliminación precisa de material para lograr un ajuste o forma específica.

## Principio de funcionamiento

La lima funciona por cizallamiento. Cada diente en la lima actúa como una pequeña hoja que corta una viruta del material que se está trabajando.

- **Dirección**: La mayoría de las limas cortan solo en el movimiento hacia adelante (empuje). Arrastrarla hacia atrás desafila los dientes.
- **Dureza**: La lima debe ser más dura que el material que está limando.

## Cómo crearla

### Pasos de fabricación

1.  **Forjado**: Dar forma a una pieza de acero con alto contenido de carbono en el perfil deseado (plana, redonda, media caña).
2.  **Recocido**: Ablandar el acero para que se puedan cortar los dientes.
3.  **Rectificado**: Rectificar la superficie para que quede plana y lisa.
4.  **Picado**: Usar un cincel afilado y duro para golpear la pieza de lima en ángulo, levantando una rebaba (el diente). Esto se hace repetida y precisamente a lo largo de la longitud.
5.  **Endurecido**: Este es el paso más crítico. La lima se calienta a temperatura crítica y se templa (generalmente en salmuera) para hacerla extremadamente dura. A menudo se deja sin revenir o solo ligeramente revenida para maximizar la dureza.

## Materiales necesarios

- **Materiales esenciales**:
  - **Acero de alto carbono**: Esencial para mantener un borde afilado y ser más duro que otros metales.
- **Herramientas**:
  - **Cincel**: Un cincel corto y duro especial para cortar los dientes.
  - **Martillo**: Para golpear el cincel.

## Variantes y mejoras

- **Escofina**: Tiene dientes elevados individuales, utilizados para madera y materiales blandos (eliminación más gruesa).
- **Picado sencillo**: Filas de dientes paralelos (acabado más suave).
- **Picado doble**: Dos conjuntos de filas diagonales que se cruzan (eliminación más rápida).
- **Limas de aguja**: Limas muy pequeñas para trabajos de precisión (joyería, relojería).

## Límites y riesgos

- **Frágil**: Debido a que son muy duras, las limas son frágiles y pueden romperse si se usan como palanca o se golpean.
- **Embotamiento (Pinning)**: Los metales blandos (como aluminio o cobre) pueden obstruir los dientes. Se necesita una carda (cepillo) para limpiarla.
- **Óxido**: El acero con alto contenido de carbono se oxida fácilmente.
