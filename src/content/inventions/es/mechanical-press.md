# Breve descripción

Una prensa mecánica es una máquina herramienta que cambia la forma de una pieza de trabajo mediante la aplicación de presión. Es fundamental para la fabricación, permitiendo el conformado preciso y consistente de materiales, particularmente metales.

## Uso / Función

- **Forja:** Dar forma al metal mediante fuerza de compresión.
- **Estampado:** Cortar o estampar láminas de metal planas.
- **Doblado:** Deformar material en un ángulo o forma deseada.
- **Cizallado:** Cortar material usando una cuchilla impulsada por la prensa.
- **Escala:** Desde pequeñas prensas de volante de taller hasta prensas industriales masivas.

## Principio de funcionamiento

La prensa mecánica transforma el movimiento rotacional (a menudo de un motor o volante manual) en movimiento lineal para impulsar un "ariete" o "corredera" contra una pieza de trabajo colocada sobre una "cama" estacionaria.
- **Almacenamiento de energía:** Un volante almacena energía cinética rotacional.
- **Transmisión:** Un mecanismo (tornillo, cigüeñal, eje excéntrico o palanca) convierte la rotación en un golpe lineal potente.
- **Impacto/Presión:** El ariete entrega la energía almacenada a la pieza de trabajo de manera controlada.

## Cómo crearla

1.  **Construcción del marco:** Construir un marco rígido y pesado (en forma de C o de O) que pueda soportar altas fuerzas de reacción sin deformarse.
2.  **Mecanismo de accionamiento:**
    - **Prensa de tornillo:** Instalar un gran tornillo vertical a través de un collar roscado en el marco. Unir una manija o volante a la parte superior.
    - **Prensa de cigüeñal:** Montar un cigüeñal y una biela para impulsar el ariete hacia arriba y hacia abajo.
3.  **Ariete y cama:** Instalar un bloque deslizante guiado (ariete) en la parte inferior del mecanismo y una mesa sólida (cama) debajo de él.
4.  **Herramental:** Unir el punzón específico (al ariete) y la matriz (a la cama) para la operación deseada.

## Materiales necesarios

- **Marco:** Hierro fundido o placas de acero soldadas para rigidez; madera pesada para versiones primitivas.
- **Mecanismo:** Acero endurecido para tornillos, cigüeñales y engranajes; bronce para bujes/cojinetes.
- **Volante:** Rueda de hierro pesada para almacenar momento.
- **Herramental:** Acero para herramientas (endurecido y templado) para resistir el desgaste y la deformación.

## Variantes y mejoras

- **Prensa de tornillo:** Usa un tornillo para impulsar el ariete (ej. imprenta de Gutenberg, prensa de acuñación). Lenta pero de gran fuerza.
- **Prensa de volante:** Una prensa de tornillo con un volante contrapesado para impacto.
- **Prensa de cigüeñal/excéntrica:** Usa un eje giratorio para impulsar el ariete. Más rápida, usada para punzonado/corte.
- **Prensa de articulación (Knuckle-Joint):** Usa un mecanismo de palanca para una fuerza muy alta en la parte inferior del golpe (acuñación).
- **Prensa hidráulica:** Usa presión de fluido en lugar de enlaces mecánicos (variante moderna).

## Límites y riesgos

- **Peligro de aplastamiento:** Extremadamente peligroso para manos y dedos. Requiere guardas de seguridad.
- **Falla del marco:** Si la prensa se sobrecarga, el marco puede agrietarse o romperse.
- **Límite de energía:** Las prensas mecánicas dependen de la energía almacenada en el volante; si el corte es demasiado duro, la prensa puede detenerse.
