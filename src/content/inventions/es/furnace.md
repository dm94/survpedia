# Breve descripción

Un horno de fundición es un dispositivo utilizado para el calentamiento a altas temperaturas. A diferencia de un **[Horno](/es/inventions/kiln)** que cuece o seca materiales (como cerámica), un horno de fundición se utiliza típicamente para fundir metales, fundir vidrio o quemar combustible para generar calor para otros procesos.

## Uso / Función

- **Fundición:** Extraer metal del mineral (por ejemplo, hierro, cobre).
- **Fusión:** Licuar metales para colada (horno de fundición).
- **Fabricación de vidrio:** Fundir arena y otros ingredientes para hacer vidrio.
- **Calefacción:** Proporcionar calor para un edificio o máquina de vapor.

## Principio de funcionamiento

Los hornos operan conteniendo un fuego extremadamente caliente dentro de una cámara refractaria (resistente al calor).

- **Combustible:** Utiliza combustibles de alta energía como **[Carbón Vegetal](/es/materials/charcoal)**, hulla o coque.
- **Inyección de Aire:** La mayoría de los hornos de alta temperatura (como los altos hornos) utilizan **[Fuelles](/es/inventions/bellows)** para bombear aire (oxígeno), aumentando la tasa de combustión y la temperatura significativamente más allá de lo que puede lograr un tiro natural.

## Cómo crearlo

1.  **Revestimiento Refractario:** Construir un eje vertical o cámara utilizando materiales resistentes al calor (arcilla refractaria, arenisca, ladrillo refractario). La piedra ordinaria puede explotar o desmoronarse.
2.  **Entrada de Aire (Tobera):** Instalar un tubo o boquilla en la parte inferior para inyectar aire desde los fuelles.
3.  **Carga:** Cargar combustible y mineral en capas alternas desde la parte superior (para hornos de cuba).
4.  **Sangrado:** Crear un agujero en la parte inferior para extraer el metal fundido y la escoria.

## Materiales necesarios

- **Estructura:** **[Arcilla](/es/materials/clay)** mezclada con arena/chamota, o Ladrillo Refractario.
- **Combustible:** **[Carbón Vegetal](/es/materials/charcoal)** es el estándar histórico; Coque para altos hornos modernos.
- **Fundente:** Piedra caliza (para eliminar impurezas/escoria).

## Variantes y mejoras

- **Fragua Catalana (Bloomery):** Horno de hierro primitivo que produce una esponja sólida de hierro (lupa), no líquido.
- **Alto Horno:** Horno de cuba alta con aire forzado que funde el hierro completamente (hierro fundido).
- **Horno de Cubilote:** Utilizado para refundir hierro fundido.
- **Horno de Reverbero:** El combustible y el material están separados; el calor se refleja desde el techo sobre el material.

## Límites y riesgos

- **Temperatura:** Alcanzar el punto de fusión del hierro (1538°C) es muy difícil sin fuelles y combustible eficientes.
- **Explosiones:** El metal fundido en contacto con el agua causa explosiones de vapor.
- **Monóxido de Carbono:** La combustión incompleta produce gas mortal.

### Inventos relacionados

- **[Horno](/es/inventions/kiln):** El primo de menor temperatura.
- **[Fuelle](/es/inventions/bellows):** Esencial para altas temperaturas.
- **[Fragua](/es/inventions/forge):** Hogar abierto para calentar metal para trabajar (no fundir).
- **[Yunque](/es/inventions/anvil):** Utilizado para trabajar el metal después de calentarlo.

### Materiales relacionados

- **[Hierro](/es/materials/iron):** Producto principal.
- **[Vidrio](/es/materials/glass):** Otro producto.
- **[Carbón Vegetal](/es/materials/charcoal):** Combustible.
- **[Lana Mineral](/es/materials/mineral-wool):** Hecha en un horno de cubilote.
- **[Perlita](/es/materials/perlite):** Expandida en un horno.
- **[Vermiculita](/es/materials/vermiculite):** Exfoliada en un horno.
