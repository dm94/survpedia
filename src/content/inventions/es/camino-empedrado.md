# Breve descripción

Una superficie de camino duradera hecha de piedras de río redondeadas (guijarros) o piedras toscamente formadas incrustadas en arena o mortero. Proporciona una superficie dura y resistente a la intemperie para el tráfico pesado y el drenaje.

## Uso / Función

- **Uso principal:** Transporte de personas, animales y carros sobre una superficie duradera que no se convierte en barro.
- **Usos secundarios:** Pavimentación urbana, prevención de la erosión del suelo, definición de calles de la ciudad.
- **Escala:** Desde senderos de jardín hasta calles de ciudades y carreteras.

## Principio de funcionamiento

- **Entrelazado:** Las piedras se colocan muy juntas para que se apoyen entre sí.
- **Fricción:** La textura proporciona agarre para cascos y ruedas.
- **Drenaje:** Los espacios entre las piedras (a menudo llenos de arena) permiten que el agua drene, o la curvatura (peralte) desvía el agua hacia los lados.
- **Transferencia de carga:** Las piedras transfieren cargas puntuales pesadas de las ruedas a la base más blanda debajo sin hundirse.

## Cómo crearlo

1.  **Preparación:** Excavar el suelo hasta una profundidad estable (retirar la capa superior del suelo).
2.  **Base:** Colocar una capa de piedra triturada o grava para drenaje y estabilidad. Compactar bien con un pisón.
3.  **Cama:** Esparcir una capa de arena gruesa o gravilla (unos 5-10 cm).
4.  **Selección:** Seleccionar piedras de tamaño similar (guijarros) o forma (adoquines).
5.  **Colocación:** Colocar las piedras en posición vertical (de pie) en la cama de arena, empaquetándolas firmemente. Usar un martillo para golpearlas y colocarlas en su lugar.
6.  **Apisonado:** Usar un pisón pesado para golpear las piedras hacia abajo en la cama hasta que estén niveladas.
7.  **Rejuntado:** Barrer arena fina o grava en las juntas entre las piedras. Regar para asentarlo. Repetir hasta que las juntas estén llenas.

## Materiales necesarios

- **Esenciales:**
  - **Piedras:** Guijarros de río (redondeados) o adoquines de cantera (toscamente cúbicos).
  - **Arena:** Para la capa de cama.
  - **Grava:** Para el drenaje de la subbase.
- **Herramientas:**
  - **Pico y Pala:** Para excavación.
  - **Pisón:** Para compactar la base y las piedras.
  - **Martillo/Mazo:** Para ajustar piedras individuales.
  - **Escoba:** Para barrer arena en las juntas.

## Variantes y mejoras

- **Pavimento de piedra seca:** Piedras colocadas en arena/tierra (permeable).
- **Pavimento con mortero:** Piedras colocadas en mortero de cal o cemento (impermeable, más fuerte).
- **Adoquín belga:** Uso de piedras rectangulares labradas (adoquines) para un viaje más suave que los guijarros redondeados.
- **Pavimento con patrón:** Disponer las piedras en abanicos o arcos para mayor resistencia y estética.

## Límites y riesgos

- **Ruido/Vibración:** Puede ser muy ruidoso y áspero para vehículos con ruedas en comparación con el asfalto.
- **Resbaladizo cuando está mojado:** Las piedras de río lisas pueden ser resbaladizas para caballos o peatones.
- **Mantenimiento:** Las malas hierbas pueden crecer en las juntas de arena; las piedras pueden aflojarse si la base se lava.
- **Mano de obra:** Extremadamente laborioso de colocar piedra por piedra.
