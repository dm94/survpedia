# Breve descripción

Los contenedores son objetos huecos diseñados para transportar, organizar o conservar materiales sólidos o líquidos.

## Uso / Función

- **Almacenamiento:** Guardar excedentes de comida, semillas o agua.
- **Transporte:** Facilitar el traslado de objetos recolectados.
- **Cocción:** Permitir el hervido de alimentos (si son resistentes al calor).
- **Escala:** Doméstica y colectiva.

## Principio de funcionamiento

Crean una barrera física que agrupa elementos dispersos y, en el caso de líquidos, utilizan materiales impermeables o sellados para evitar filtraciones.

## Cómo crearlo

- **Cestería:** Entrelazar fibras vegetales flexibles para crear una estructura rígida o semirrígida.
- **Sacos:** Coser piezas de cuero o tela.
- **Vasijas:** Moldear arcilla y cocerla para lograr impermeabilidad y resistencia térmica.
- **Nivel técnico:** Básico a Intermedio.

## Materiales necesarios

- **Esenciales:** Mimbre, esparto, juncos, pieles de animales, arcilla.
- **Herramientas:** Punzones, agujas, manos.
- **Sustitutos:** Estómagos de animales (para agua), calabazas vaciadas, madera tallada.

## Variantes y mejoras

- **Cesta estanca:** Recubierta con resina o betún para llevar agua.
- **Anforas:** Diseñadas para transporte marítimo masivo.
- **Contenedores modulares:** Para optimizar espacio.

## Límites y riesgos

- **Peso:** El contenedor añade peso muerto al transporte.
- **Humedad:** Las cestas pueden pudrirse; la cerámica sin vidriar puede filtrar.
- **Rotura:** La cerámica es frágil; los sacos pueden rasgarse.
