# Breve descripción

El molino es un ingenio mecánico diseñado para triturar, moler o pulverizar materias sólidas, principalmente cereales, mediante el uso de energía motriz.

## Uso / Función

- **Alimentación:** Transformación de grano en harina.
- **Industria:** Trituración de minerales o pigmentos.
- **Prensado:** Extracción de aceites de aceitunas o semillas.
- **Bombeo:** Algunos molinos se adaptan para elevar agua.

## Principio de funcionamiento

Aprovecha una fuente de energía (humana, animal, eólica o hidráulica) para hacer girar una pieza móvil (muela superior) sobre una pieza fija (muela inferior). El material introducido entre ambas es triturado por la fricción y el peso.

## Cómo crearlo

1. Talla de dos piedras circulares (muelas) con surcos para facilitar la salida del producto.
2. Construcción de una estructura de madera para soportar el eje y el mecanismo de transmisión.
3. Instalación de un sistema de alimentación (tolva) para el grano.

## Materiales necesarios

- **Piedras de moler:** Piedra dura y rugosa como el granito o basalto.
- **Ejes y engranajes:** Madera resistente (encina, roble) o metal.
- **Estructura:** Madera para el armazón y las aspas/ruedas.

## Variantes y mejoras

- **Molino de mano:** Operado manualmente, escala doméstica.
- **Molino hidráulico:** Usa la fuerza de los ríos (ver entrada de Molino de agua para más detalles).
- **Molino de viento:** Usa la energía eólica.
- **Molino eléctrico:** Versión moderna de alta eficiencia.

## Límites y riesgos

- **Mantenimiento:** Las piedras pierden rugosidad y deben ser "picadas" periódicamente.
- **Seguridad:** Riesgo de atrapamiento en los engranajes o muelas.
- **Dependencia:** Los molinos de viento o agua dependen de las condiciones climáticas.
