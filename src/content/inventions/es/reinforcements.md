# Breve descripción

Los refuerzos estructurales son sistemas de soporte que estabilizan excavaciones, galerías y pozos para evitar derrumbes y mantener accesos seguros.

## Uso / Función

- **Estabilidad del terreno**: Sostener techos y paredes inestables.
- **Seguridad**: Reducir colapsos y desprendimientos durante la extracción.
- **Acceso continuo**: Mantener rutas de entrada, ventilación y evacuación.
- **Profundidad**: Permitir excavaciones más profundas y duraderas.
- **Escala**: Desde zanjas pequeñas hasta túneles extensos.

## Principio de funcionamiento

- **Transferencia de carga**: Puntales y marcos redirigen el peso hacia suelos más estables.
- **Confinamiento**: Entibados y revestimientos evitan que el material suelto se desplace.
- **Redundancia**: Múltiples apoyos distribuyen el riesgo de fallo puntual.
- **Anclaje**: Cuñas y amarres fijan las piezas para que no se desplacen.

## Cómo crearlo

1. **Evaluar el terreno**: Identificar zonas blandas, grietas y humedad.
2. **Elegir el sistema**: Puntales de madera, marcos en A, entibado con tablas o revestimiento de piedra.
3. **Preparar apoyos**: Nivelar bases y colocar cuñas para un contacto firme.
4. **Montar soportes**: Instalar postes y travesaños desde el acceso hacia el frente de trabajo.
5. **Revestir**: Añadir tablas o piedra para contener material suelto.
6. **Nivel técnico**: Básico para puntales de madera, intermedio para galerías entibadas, avanzado para conjuntos metálicos.

## Materiales necesarios

- **Esenciales**: **[Madera](/es/materials/wood)** para puntales y marcos, **[Piedra](/es/materials/piedra)** o **[Arcilla](/es/materials/arcilla)** para revestimiento, **[Hierro](/es/materials/iron)** o **[Acero](/es/materials/steel)** para grapas y refuerzos.
- **Herramientas**: Martillo, sierra, cincel, cuñas, cuerdas o fibras para amarres.
- **Sustitutos**: Tierra compactada, muros de piedra seca, amarres con **[Fibras vegetales](/es/materials/plant-fibers)**.

## Variantes y mejoras

- **Entibado de madera**: Postes y travesaños con tablas de revestimiento.
- **Cribas o jaulas**: Troncos apilados en cajones para suelos blandos.
- **Revestimiento de piedra**: Mampostería para túneles estables y duraderos.
- **Marcos metálicos**: Arcos de acero para cargas altas y larga vida útil.

## Límites y riesgos

- **Degradación**: La madera se pudre con humedad; el metal se corroe.
- **Instalación deficiente**: Un soporte mal colocado puede fallar de forma súbita.
- **Mantenimiento**: Requiere inspecciones frecuentes y reemplazo de piezas.
- **Falsa seguridad**: Refuerzos insuficientes pueden incentivar excavaciones demasiado profundas.
