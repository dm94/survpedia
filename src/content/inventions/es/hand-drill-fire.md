# Breve descripción

El fuego por taladro manual es un método de fricción que usa las manos para girar un huso de madera contra una tabla base hasta formar una brasa.

## Uso / Función

- **Uso principal:** Crear una brasa para encender fuego.
- **Usos secundarios:** Entrenar la selección de materiales y la técnica.
- **Escala:** Individual.

## Principio de funcionamiento

La rotación rápida genera calor por fricción en el punto de contacto. El polvo de madera se acumula en una muesca, se carboniza y forma una brasa que se transfiere a la yesca.

## Cómo crearlo

1. **Seleccionar maderas:** Tabla base blanda y huso algo más duro.
2. **Preparar el hueco:** Hacer una pequeña cavidad en la tabla.
3. **Cortar la muesca:** Abrir una muesca en V hacia el borde para recoger el polvo.
4. **Girar:** Rodar el huso con las palmas manteniendo presión constante.
5. **Brasa:** Seguir hasta que el polvo humee y brille, luego transferir a la yesca.
6. **Nivel técnico:** Básico.

## Materiales necesarios

- **Esenciales:** Tabla base seca, huso recto, yesca.
- **Herramientas:** Cuchillo o piedra afilada para tallar.
- **Sustitutos:** Taladro de arco para reducir esfuerzo.

## Variantes y mejoras

- **Taladro a dos personas:** Una persona gira y otra sujeta el huso.
- **Muesca marcada:** Acelera la formación de la brasa.
- **Pre-secado:** Calentar la madera cerca del fuego mejora el resultado.

## Límites y riesgos

- **Alto esfuerzo:** Cansa rápido.
- **Sensibilidad a la humedad:** La madera húmeda falla.
- **Ampollas:** La fricción prolongada puede lesionar las manos.
