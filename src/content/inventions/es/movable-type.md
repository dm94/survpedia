# Breve descripción

Los tipos móviles son piezas individuales (generalmente de metal) que llevan un carácter alfanumérico o signo de puntuación en relieve en una de sus caras. Estas piezas pueden ensamblarse para formar palabras, líneas y páginas de texto, y luego desmontarse y reutilizarse infinitas veces para componer nuevos textos.

## Uso / Función

Su función principal es permitir la composición flexible y reutilizable de textos para la impresión:

- **Composición de Textos:** Permite formar cualquier combinación de palabras y oraciones sin tener que tallar una página entera cada vez.
- **Reutilización:** Una vez impresa una página, los tipos se pueden "distribuir" (devolver a sus cajas) y usar para la siguiente.
- **Corrección:** Es fácil corregir errores tipográficos simplemente cambiando los caracteres afectados antes de imprimir.
- **Escala:** Fundamental para la producción masiva de libros y documentos.

## Principio de funcionamiento

El sistema se basa en la estandarización y la modularidad:

1.  **Uniformidad:** Todos los tipos deben tener exactamente la misma altura ("altura de papel") para que la prensa aplique presión uniforme.
2.  **Cuerpo:** El tamaño del bloque de metal (cuerpo) determina el espaciado y la alineación.
3.  **Aleación:** El uso de una aleación específica (metal de tipos) permite fundir detalles finos que se endurecen rápidamente y soportan la presión de la prensa sin deformarse inmediatamente.

## Cómo crearlo

El proceso de creación de tipos móviles (fundición) requiere gran precisión:

1.  **Tallado del Punzón:** Un artesano talla la letra en relieve en el extremo de una barra de acero duro. Este es el paso más difícil y artístico.
2.  **Creación de la Matriz:** El punzón de acero se golpea contra una barra de cobre más blanda, dejando una impresión profunda de la letra (la matriz).
3.  **Ajuste de la Matriz:** La matriz se lima y ajusta para asegurar que encaje perfectamente en el molde.
4.  **Fundición:**
    - Se coloca la matriz en un molde de mano ajustable.
    - Se vierte la aleación fundida (plomo, estaño, antimonio) en el molde.
    - Se hace un movimiento rápido para asegurar que el metal llene todos los rincones de la matriz.
    - Se abre el molde y se expulsa el tipo.
5.  **Acabado:** Se rompe el exceso de metal (la "colada") y se lija la base para que tenga la altura exacta requerida.

## Materiales necesarios

- **Metal de Tipos:** Aleación de Plomo (volumen principal), Antimonio (dureza y expansión) y Estaño (fluidez y tenacidad).
- **Punzón:** Acero endurecido.
- **Matriz:** Cobre.
- **Molde de mano:** Acero, latón y madera.
- **Cajas:** Madera para organizar los tipos (caja alta para mayúsculas, caja baja para minúsculas).

## Variantes y mejoras

- **Tipos de Madera:** Usados para letras muy grandes (carteles) donde el metal sería demasiado pesado y costoso.
- **Linotipia:** Máquina que funde líneas enteras de texto a la vez (siglo XIX), acelerando enormemente la composición.
- **Monotipia:** Máquina que funde y compone caracteres individuales automáticamente.
- **Tipografía Digital:** La evolución moderna donde los "tipos" son archivos de software.

## Límites y riesgos

- **Desgaste:** Los tipos de metal se desgastan con el uso repetido y la presión, perdiendo nitidez.
- **Toxicidad:** El contacto constante con el plomo representa un riesgo grave de salud (saturnismo) para los tipógrafos.
- **Complejidad:** Crear un juego completo de punzones y matrices para una fuente requiere años de trabajo experto.
- **Peso:** Las formas de tipos compuestas son muy pesadas y difíciles de transportar sin desordenarlas.
