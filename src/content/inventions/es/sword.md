# Breve descripción

Arma blanca cuerpo a cuerpo destinada a cortar o estocar. Consta de una hoja larga de metal y una empuñadura con guarda. Es un símbolo de guerra y estatus.

## Uso / Función

- **Combate**: Se usa para tajar, cortar o estocar a los oponentes.
- **Defensa**: Puede parar golpes entrantes.
- **Estatus**: A menudo un símbolo ceremonial de autoridad o rango.

## Principio de funcionamiento

La espada actúa como una palanca y una cuña.
- **Corte**: El filo concentra la fuerza en un área minúscula para cortar material.
- **Estocada**: La punta concentra la fuerza para perforar armaduras o carne.
- **Equilibrio**: La distribución del peso (controlada por el pomo y el estrechamiento) determina si corta mejor (peso adelante) o se maneja más rápido (peso atrás).

## Cómo crearlo

1.  **Forja**: Una barra de acero se calienta y martillea hasta darle forma de hoja, espiga y punta.
2.  **Conformado**: Se amolan los biseles para crear la geometría del filo. El **[Equilibrado](/es/knowledge/balancing)** es crítico aquí.
3.  **Tratamiento Térmico**: La hoja se endurece (templa) y reviene para que sea lo suficientemente dura para mantener el filo pero lo suficientemente tenaz para no romperse.
4.  **Montaje de la empuñadura**: Se ajusta una guarda, se añade el mango (madera/cuero) y se remacha un pomo pesado en el extremo de la espiga para asegurarla y equilibrar la espada.

## Materiales necesarios

- **Esenciales**: **[Acero](/es/materials/steel)** o **[Hierro](/es/materials/iron)** de alta calidad (o **[Bronce](/es/materials/bronze)** históricamente).
- **Empuñadura**: **[Madera](/es/materials/wood)**, **[Cuero](/es/materials/leather)**, hueso o alambre de metal.
- **Herramientas**: **[Fragua](/es/inventions/forge)**, **[Yunque](/es/inventions/anvil)**, **[Martillo](/es/inventions/hammer)**, **[Piedra de amolar](/es/inventions/hand-mill)**.

## Variantes y mejoras

- **Gladius**: Espada corta de estocada.
- **Espada Larga (Longsword)**: Espada de dos manos para alcance y potencia.
- **Katana**: Espada curva de un solo filo optimizada para el corte.
- **Estoque**: Espada delgada y puntiaguda optimizada para estocar y batirse en duelo.

## Límites y riesgos

- **Mantenimiento**: Se oxida fácilmente si no se aceita; los filos se desafilan rápidamente.
- **Habilidad**: Requiere un entrenamiento significativo para usarla eficazmente sin lesionarse.
- **Fragilidad**: Un mal tratamiento térmico puede hacer que la hoja se rompa con el impacto.
