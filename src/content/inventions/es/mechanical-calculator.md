# Breve descripción

Un dispositivo mecánico utilizado para realizar las operaciones básicas de aritmética de forma automática. Reemplaza el cálculo mental o herramientas manuales como el ábaco con un sistema de engranajes y palancas interconectados que rastrean valores numéricos.

## Uso / Función

- **Uso principal:** Realizar sumas, restas, multiplicaciones y divisiones con alta precisión.
- **Usos secundarios:** Contabilidad, navegación (cálculos logarítmicos) y procesamiento de datos científicos.
- **Escala:** Dispositivos de escritorio para empresas, ingenieros y científicos.

## Principio de funcionamiento

Las calculadoras mecánicas se basan en mecanismos como el "tambor escalonado" o la "rueda de pasadores" para representar y manipular números:

1.  **Entrada:** Los números se establecen mediante deslizadores, teclas o ruedas.
2.  **Suma:** Al girar una manivela se rotan los engranajes. Un mecanismo de "acarreo" asegura que cuando un engranaje completa una rotación completa (representando 0-9), avanza el siguiente engranaje (la posición de las decenas) un diente.
3.  **Resta:** Se realiza invirtiendo la rotación o utilizando la suma complementaria.
4.  **Multiplicación/División:** Se logra mediante sumas o restas repetidas combinadas con el desplazamiento del carro (cambiando la posición decimal).

## Cómo crearlo

Una calculadora mecánica es uno de los dispositivos mecánicos más complejos de construir, requiriendo una precisión extrema.

1.  **Ruedas de dígitos:** Una serie de cilindros o discos marcados del 0 al 9.
2.  **Mecanismo de acarreo:** La parte más difícil; una palanca o leva de "acarreo de decenas" que activa la siguiente rueda de dígitos exactamente en el momento adecuado sin atascar el sistema.
3.  **Acumulador:** Un registro que muestra el resultado de los cálculos.
4.  **Mecanismo de puesta a cero:** Una forma de devolver rápidamente todas las ruedas de dígitos a cero.
5.  **Carcasa:** Una caja robusta, a menudo de **[Madera](/es/materials/wood)** o metal, para proteger los delicados trabajos internos.

## Materiales necesarios

- **Engranajes y mecanismos:** Se prefiere el **[Latón](/es/materials/brass)** por su facilidad de mecanizado y baja fricción. El **[Acero](/es/materials/steel)** se utiliza para piezas de alto desgaste como resortes y trinquetes.
- **Ejes:** Varillas de **[Acero](/es/materials/steel)** pulido.
- **Bastidor:** Placas de **[Hierro](/es/materials/iron)** o **[Latón](/es/materials/brass)** para mantener los ejes en perfecta alineación.
- **Lubricación:** Aceite vegetal fino o grasas animales para asegurar un funcionamiento suave.

## Variantes y mejoras

- **Pascalina:** Una de las primeras calculadoras (siglo XVII), principalmente para sumas y restas.
- **Aritmómetro:** La primera calculadora mecánica comercialmente exitosa, capaz de realizar las cuatro operaciones aritméticas.
- **Comptómetro:** Una calculadora impulsada por teclas donde el acto de presionar las teclas proporciona la energía para el cálculo.
- **Máquina diferencial:** Una calculadora mecánica masiva y especializada diseñada para tabular funciones polinómicas (precursora de la computadora mecánica).

## Límites y riesgos

- **Precisión:** Un pequeño error en la forma de un diente de engranaje puede hacer que toda la máquina se atasque o proporcione resultados incorrectos.
- **Complejidad:** Extremadamente difícil de reparar si una pieza interna se rompe.
- **Velocidad:** Mucho más lenta que las calculadoras electrónicas; las multiplicaciones complejas pueden requerir un esfuerzo físico significativo.
- **Tamaño:** Las máquinas capaces suelen ser pesadas y voluminosas.
