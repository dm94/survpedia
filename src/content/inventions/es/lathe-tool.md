# Breve descripción

El buril es una herramienta de corte de un solo punto utilizada en **[Tornos](/es/inventions/lathe)** para mecanizar materiales. A diferencia de las herramientas rotativas como **[Brocas](/es/inventions/drill-bit)** o **[Fresas](/es/inventions/milling-cutter)**, el buril suele mantenerse estacionario mientras la pieza de trabajo gira, eliminando material en forma de viruta continua.

## Uso / Función

- **Cilindrado**: Reducir el diámetro de una pieza cilíndrica.
- **Refrentado**: Aplanar el extremo de una pieza.
- **Roscado**: Cortar roscas de tornillo internas o externas.
- **Tronzado**: Cortar una pieza para separarla del material en bruto.

## Principio de funcionamiento

El buril tiene un borde cortante afilado con ángulos específicos (incidencia, ataque) que permiten que penetre en el material giratorio y cizalle una viruta sin rozar excesivamente.

- **Ángulo de ataque**: Facilita el flujo de la viruta.
- **Ángulo de incidencia**: Evita que la herramienta roce contra la pieza recién cortada.

## Cómo crearla

### Versión básica (Acero al Carbono)

1.  **Forjado**: Se da forma a una barra de acero.
2.  **Afilado**: Se usan limas o piedras de afilar para crear los ángulos de corte.
3.  **Templado**: Se endurece la punta para resistir el desgaste.

### Versión estándar (Acero Rápido - HSS)

Se parte de una barra cuadrada de HSS (cobalto o tungsteno) y se afila en una **[Muela de esmeril](/es/inventions/grinding-wheel)** para darle la geometría deseada.

### Versión avanzada (Carburo)

Se suelda una pequeña plaquita de **[Carburo de Tungsteno](/es/materials/tungsten)** en un vástago de acero, o se utilizan plaquitas intercambiables. Esto permite cortar materiales más duros a mayor velocidad.

## Materiales necesarios

- **Esenciales**:
  - **Acero para herramientas**: Acero al carbono (básico) o HSS (estándar).
  - **Carburo**: Para alto rendimiento.
- **Herramientas**: **[Muela de esmeril](/es/inventions/grinding-wheel)** (esencial para dar forma y afilar).

## Variantes y mejoras

- **Herramienta de desbaste**: Robusta, para eliminar mucho material rápidamente.
- **Herramienta de acabado**: Punta más aguda o redondeada para dejar una superficie suave.
- **Barra de mandrinar**: Para agrandar agujeros interiores.
- **Herramienta de tronzar**: Hoja delgada para cortar la pieza.

## Límites y riesgos

- **Altura del centro**: La punta debe estar exactamente a la altura del eje del torno. Si está muy alta o baja, no cortará bien o vibrará.
- **Voladizo**: Si la herramienta sobresale demasiado del portaherramientas, vibrará (chatter) y dejará un mal acabado.
- **Calor**: El calor excesivo puede ablandar el acero HSS (quemar la punta).
