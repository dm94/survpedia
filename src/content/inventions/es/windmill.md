# Breve descripción

Un molino de viento es una estructura que convierte la energía cinética del viento en energía rotacional mediante aspas o velas. Esta energía se utiliza para moler grano, bombear agua o impulsar otra maquinaria.

## Uso / Función

- **Molienda:** Triturar grano para hacer harina.
- **Bombeo:** Drenar humedales o extraer agua para riego.
- **Aserrado:** Cortar madera en aserraderos.
- **Industrial:** Triturar semillas para aceite, procesar especias o batanar telas.
- **Escala:** Industrial / Comunidad local.

## Principio de funcionamiento

1.  **Captura:** El viento empuja contra las aspas o velas inclinadas, creando sustentación y arrastre que hace girar el eje del viento.
2.  **Transmisión:** El eje del viento (horizontal o ligeramente inclinado) conecta con una gran rueda de freno.
3.  **Engranajes:** La rueda de freno impulsa una linterna (engranaje) en un eje principal vertical, convirtiendo el movimiento de horizontal a vertical.
4.  **Aplicación:** El eje vertical impulsa las muelas (a través de una rueda de espuelas) o una rueda de paletas/tornillo de Arquímedes para bombear.

## Cómo crearlo

1.  **Cimientos:** Construir una base estable (ladrillo, piedra o caballete de madera) para soportar el peso y resistir las fuerzas del viento.
2.  **Estructura:** Construir la torre o el cuerpo. En un "Molino de poste", todo el cuerpo gira sobre un poste central. En un "Molino de torre", solo gira la caperuza.
3.  **Aspas:** Construir el armazón para las aspas (generalmente 4) de madera ligera y resistente. Cubrir con velas de **tela** (lona) que se puedan ajustar según la velocidad del viento.
4.  **Mecanismo:** Instalar el eje del viento, la rueda de freno, el eje vertical y los engranajes. Usar madera dura (como roble) o hierro.
5.  **Muelas:** Instalar la solera (fija) y la volandera (giratoria).
6.  **Orientación:** Añadir un palo de cola o un abanico (fantail) para girar las aspas hacia el viento manual o automáticamente.

## Materiales necesarios

- **Estructura:** Madera resistente (roble) o mampostería (ladrillo/piedra) para la torre.
- **Aspas:** Entramado de madera ligera y **tela** fuerte (lona) o contraventanas de madera.
- **Engranajes:** Madera dura (manzano, carpe) o hierro fundido para mayor durabilidad.
- **Muelas:** Piedra dura y abrasiva (piedra de molino, granito).
- **Cuerda:** Para controlar las velas y el freno.

## Variantes y mejoras

- **Molino de poste:** El tipo más antiguo; todo el edificio gira para enfrentar el viento. Tamaño limitado.
- **Molino de torre:** Solo gira la caperuza superior. Permite un edificio más grande y estable.
- **Abanico (Fantail):** Un pequeño ventilador en la parte trasera que gira automáticamente la caperuza hacia el viento.
- **Velas de resorte/patente:** Contraventanas que se abren/cierran automáticamente para regular la velocidad, reemplazando el manejo manual de la lona.

## Límites y riesgos

- **Intermitencia:** No funciona sin viento; inútil en tiempo calmo.
- **Tormentas:** Los vientos fuertes pueden destruir las aspas o incendiar el molino (fuego por fricción) si falla el freno.
- **Mantenimiento:** Las velas se desgastan rápidamente; los engranajes necesitan engrase constante (grasa animal).
- **Seguridad:** Las aspas giratorias son pesadas y rápidas; ser golpeado es fatal. El fallo del freno puede llevar a una velocidad descontrolada.
