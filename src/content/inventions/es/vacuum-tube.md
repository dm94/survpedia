# Breve descripción

Una válvula de vacío (o tubo de vacío) es un componente electrónico que controla el flujo de corriente eléctrica en un alto vacío entre electrodos a los que se aplica una diferencia de potencial. Fue el componente clave de la electrónica temprana (radio, TV, computadoras).

## Uso / Función

-   **Amplificación:** Aumenta la potencia de una señal débil (audio, radio).
-   **Conmutación:** Actúa como un interruptor rápido encendido/apagado para lógica digital (computadoras antiguas).
-   **Rectificación:** Convierte corriente alterna (CA) en corriente continua (CC).
-   **Oscilación:** Genera señales de radiofrecuencia.

## Principio de funcionamiento

Se basa en la **emisión termiónica**:
1.  **Cátodo:** Un filamento (tungsteno) se calienta hasta que emite electrones.
2.  **Vacío:** La ampolla de vidrio se vacía para que las moléculas de aire no obstruyan los electrones.
3.  **Ánodo (Placa)::** Una placa cargada positivamente atrae los electrones, creando un flujo de corriente.
4.  **Rejilla:** Una malla colocada entre el cátodo y el ánodo controla el flujo. Un pequeño cambio de voltaje en la rejilla causa un gran cambio de corriente en el ánodo (amplificación).

## Cómo crearlo

1.  **Ampolla:** Soplar una bombilla de vidrio.
2.  **Electrodos:** Fabricar el cátodo (filamento), rejilla (alambre fino) y ánodo (placa) de metal.
3.  **Montaje:** Montar los electrodos en un vástago, asegurando que no se toquen.
4.  **Sellado:** Insertar el montaje en la bombilla y sellar el vidrio.
5.  **Evacuación:** Usar una bomba de vacío para extraer el aire a través de un tubo pequeño, luego sellarlo. A menudo se "flashea" un "getter" (metal reactivo) dentro para absorber el gas restante.

## Materiales necesarios

-   **Vidrio:** Para la envoltura hermética.
-   **Tungsteno:** Para el filamento calentador (alto punto de fusión).
-   **Metales:** Níquel o acero para la placa y la rejilla; Cobre para los conectores.
-   **Vacío:** No es un material, pero es una condición esencial.

## Variantes y mejoras

-   **Diodo:** Dos electrodos (rectificación).
-   **Triodo:** Tres electrodos (amplificación - el gran avance).
-   **Pentodo:** Cinco electrodos (mejor eficiencia y estabilidad).
-   **Miniaturización:** Tubos más pequeños para radios portátiles antes de los transistores.

## Límites y riesgos

-   **Calor:** Se calientan mucho y desperdician energía.
-   **Fragilidad:** El vidrio se puede romper fácilmente.
-   **Voltaje:** Requieren altos voltajes para operar.
-   **Vida útil:** Los filamentos eventualmente se queman.
