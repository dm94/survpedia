# Breve descripción

El Sonar (Sound Navigation and Ranging) es una técnica que utiliza la propagación del sonido (generalmente bajo el agua) para navegar, comunicarse o detectar objetos sobre o bajo la superficie del agua. Es el equivalente submarino del radar.

## Uso / Función

- **Navegación Marítima**: Medición de la profundidad del agua (batimetría) y detección de obstáculos submarinos.
- **Pesca**: Localización de bancos de peces (Ecosondas).
- **Guerra de Submarinos**: Detección y seguimiento de otros barcos.
- **Oceanografía**: Mapeo del fondo marino.
- **Comunicación**: Envío de señales acústicas entre barcos submarinos.

## Principio de funcionamiento

El sonar se basa en el hecho de que el sonido viaja mucho más eficiente y rápido en el agua (aprox. 1.500 m/s) que en el aire.

1.  **Generación de Pulso**: Se genera una señal electrónica.
2.  **Transducción**: Un transductor convierte la señal eléctrica en una onda de sonido (un "ping").
3.  **Propagación**: La onda de sonido viaja a través del agua.
4.  **Reflexión**: La onda golpea un objeto (el fondo, un pez, un submarino) y rebota.
5.  **Recepción**: El transductor (o un hidrófono) captura el eco que regresa.
6.  **Cálculo**: El tiempo transcurrido entre el ping y el eco determina la distancia.

## Cómo crearlo

- **Transductor**: Un dispositivo hecho de materiales piezoeléctricos (como el cuarzo o cerámicas especializadas) que cambia de forma cuando se aplica voltaje, creando sonido.
- **Hidrófono**: Un micrófono especializado para uso bajo el agua.
- **Amplificador**: Circuitos electrónicos para aumentar las débiles señales que regresan.
- **Pantalla**: Una pantalla o medidor para mostrar la profundidad o el objeto detectado.
- **Nivel técnico**: Avanzado.

- **Materiales esenciales**:

## Variantes y mejoras

- **Sonar Activo**: Envía una señal y escucha el eco (revela tu propia posición).
- **Sonar Pasivo**: Solo escucha los sonidos emitidos por otros objetos (sigiloso).
- **Sonar de Barrido Lateral**: Remolcado detrás de un barco para crear una imagen de área amplia del fondo marino.
- **Sonar Multihaz**: Utiliza múltiples haces de sonido para mapear una amplia franja del fondo del océano con gran detalle.

## Límites y riesgos

- **Termoclinas**: Capas de agua con diferentes temperaturas pueden desviar las ondas de sonido, creando "puntos ciegos".
- **Salinidad y Presión**: Afectan la velocidad del sonido y pueden distorsionar las lecturas.
- **Impacto Ambiental**: El sonar activo muy fuerte puede desorientar o herir a los mamíferos marinos como ballenas y delfines.
- **Alcance**: El sonido eventualmente se disipa, limitando hasta dónde puede "ver" el sonar.
