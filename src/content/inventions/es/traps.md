# Breve descripción

Las trampas son dispositivos mecánicos diseñados para capturar o matar animales mediante un disparador que libera energía almacenada o un peso en caída.

## Uso / Función

- **Captura de alimento:** Obtener caza menor rica en proteína y grasa.
- **Control de plagas:** Proteger cultivos, almacenes y estructuras.
- **Protección de apiarios:** Reducir presión de roedores y depredadores cerca de la **[Apicultura](/es/knowledge/beekeeping)**.
- **Pesca:** Embudos, estacadas o canastas para captura pasiva en ríos y mareas.
- **Defensa y alarma:** Disuadir intrusos o señalar movimiento.
- **Escala:** Uso individual, familiar o de pequeño grupo.

## Principio de funcionamiento

Las trampas combinan un disparador con energía almacenada o gravedad. El animal activa el disparador y libera un lazo, mandíbulas o una caída. La colocación guía el movimiento del animal hacia el punto de captura y el camuflaje reduce la evasión.

## Cómo crearla

1. **Define el objetivo:** Identifica el animal, su tamaño y rutas de paso.
2. **Elige el tipo de trampa:** Lazo, caída, foso o jaula.
3. **Construye el disparador:** Usa una clavija equilibrada, muesca o palanca que se libere con poca presión.
4. **Coloca la trampa:** Sitúala en senderos estrechos o entradas de madriguera y guía el paso con palos o piedras.
5. **Camufla y estabiliza:** Reduce el olor humano y evita que se mueva.
6. **Revisa con frecuencia:** Evita sufrimiento, pérdida y reinicia si es necesario.

## Materiales necesarios

- **Esenciales:** Armazón de **[Madera](/es/materials/wood)**, peso de **[Piedra](/es/materials/piedra)**, **[Fibras vegetales](/es/materials/plant-fibers)** para cordaje, tiras de **[Cuero](/es/materials/leather)**, ganchos o mandíbulas de **[Hierro](/es/materials/iron)**.
- **Herramientas:** Cuchillo, maza, pala.
- **Sustitutos:** Ramas flexibles como muelle, **[Alambre](/es/inventions/wire)** para lazos, **[Cuerda](/es/inventions/rope)** para líneas y anclajes.

## Variantes y mejoras

- **Trampa de lazo:** El lazo se cierra sobre la pata o el cuello.
- **Trampa de caída:** Un peso cae al liberarse el disparador.
- **Trampa de foso:** Foso oculto con estacas para animales grandes.
- **[Pozo para urogallos](/es/inventions/grouse-pit):** Foso poco profundo para aves terrestres.
- **Trampa de jaula:** Puerta que se cierra para captura viva.
- **[Trampa para ratones](/es/inventions/mouse-trap):** Trampa pequeña de golpe o caja para roedores.
- **[Atrapamoscas](/es/inventions/fly-catcher):** Trampa adhesiva o de embudo para moscas.
- **[Trampa Kulema](/es/inventions/kulema-trap):** Trampa de caja con muelle para pequeños depredadores.
- **[Trampa de corral para lobos](/es/inventions/wolf-pen-trap):** Corral grande con embudo y puerta de una sola dirección.
- **[Lazo de tensión](/es/inventions/tension-snare):** Una rama doblada eleva la presa para evitar fugas.
- **Weir de pesca:** Estacas y ramas guían peces hacia un corral.
- **Canasta trampa:** Cesta con embudo y garganta estrecha que retiene peces.

## Límites y riesgos

- **Captura no deseada:** Puede afectar a mascotas o fauna protegida.
- **Seguridad:** Los muelles y pesos pueden lesionar al constructor.
- **Clima:** Lluvia y viento pueden disparar o inutilizar la trampa.
- **Regulación:** En muchas zonas hay restricciones o prohibiciones.
