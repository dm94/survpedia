# Breve descripción

Un componente circular diseñado para girar sobre un eje. La rueda es uno de los componentes clave de la rueda y el eje, que es una de las seis máquinas simples. Permite mover objetos pesados fácilmente facilitando el movimiento o transporte mientras soporta una carga, o realizando trabajo en máquinas.

## Uso / Función

- **Transporte:** Fundamental para carros, vagones y [carros de guerra](/es/inventions/chariot).
- **Maquinaria:** Esencial para engranajes, volantes de inercia, [ruedas hidráulicas](/es/inventions/water-wheel) y [tornos de alfarero](/es/inventions/potters-wheel).
- **Conversión de energía:** Transforma el movimiento lineal en movimiento rotatorio y viceversa.

## Principio de funcionamiento

La rueda reduce la fricción facilitando el movimiento al rodar junto con el uso de un [eje](/es/inventions/axle).

- **Fricción de rodadura:** La fricción de rodadura es significativamente menor que la fricción de deslizamiento.
- **Palanca:** Una rueda es una palanca continua, con el punto de apoyo en el punto de contacto con el suelo.

## Cómo crearla

### Rueda de Madera Maciza (Básica)

1.  **Selección:** Encuentra un tronco grande de madera dura (roble, olmo).
2.  **Corte:** Corta una rodaja (disco) del tronco.
3.  **Forma:** Asegúrate de que sea circular.
4.  **Perforación:** Taladra un agujero en el centro exacto para el [eje](/es/inventions/axle).
5.  **Refuerzo:** Usa tablones cruzados o bandas de metal para evitar que se parta.

### Rueda de Radios (Avanzada)

1.  **Cubo:** Crea un cubo central con mortajas para los radios.
2.  **Llanta:** Dobla [madera](/es/materials/wood) (usando vapor) o ensambla segmentos curvos (pinas) para formar un círculo.
3.  **Radios:** Inserta los radios conectando el cubo a la llanta.
4.  **Neumático/Banda:** Ajusta por contracción una banda de [metal](/es/materials/metal) ([hierro](/es/materials/iron) o [bronce](/es/materials/bronze)) alrededor de la llanta para mantener todo unido y proporcionar durabilidad.

## Materiales necesarios

- **Madera:** Madera dura curada para mayor durabilidad.
- **Metal:** Hierro o bronce para la banda y el refuerzo del cubo.
- **Herramientas:** Sierra, cincel, taladro, martillo.

## Variantes y mejoras

- **Torno de alfarero:** Una rueda horizontal utilizada para dar forma a la arcilla.
- **Rueda hidráulica:** Una rueda con palas o cangilones para aprovechar la energía del agua.
- **Engranaje:** Una rueda dentada para transmitir par.
- **Polea:** Una rueda con un borde acanalado para una cuerda o cadena.

## Límites y riesgos

- **Terreno:** Las ruedas requieren superficies relativamente lisas para ser efectivas.
- **Fricción:** El cojinete del eje genera fricción y calor; la lubricación es esencial.
- **Mantenimiento:** Las ruedas de radios son complejas y pueden romperse bajo estrés; las ruedas macizas son pesadas y pueden partirse.
