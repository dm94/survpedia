# Descripción breve

Una jeringa es una bomba recíproca simple que consiste en un émbolo que se ajusta firmemente dentro de un tubo cilíndrico llamado cilindro. Se utiliza para inyectar o extraer fluidos, y es esencial para la administración precisa de medicamentos, vacunas y muestreo científico.

## Uso / Función

Su propósito práctico:
- Administración precisa de medicamentos líquidos y vacunas.
- Extracción de sangre u otros fluidos para análisis.
- Irrigación de heridas u orificios.
- Aplicación de precisión de lubricantes o adhesivos en trabajos técnicos.

## Principio de funcionamiento

La jeringa funciona según el principio de presión atmosférica y vacío. Cuando se tira del émbolo hacia atrás, se crea un vacío dentro del cilindro, lo que aspira el líquido a través del orificio. Cuando se empuja el émbolo hacia adelante, se crea una presión que obliga al líquido a salir. El sellado hermético entre el émbolo y el cilindro (a menudo ayudado por una junta de goma) es crucial para mantener esta diferencia de presión.

## Cómo crearlo

- **Versión funcional mínima**: Un tubo hueco (como un tubo de vidrio) con un pistón de madera o metal envuelto en cuero o caucho para crear un sello.
- **Esencial**: Un cilindro perfectamente cilíndrico y liso, y un émbolo que forme un sello hermético.
- **Nivel técnico**: Alto (requiere mecanizado de precisión o soplado de vidrio para asegurar un ajuste perfecto y evitar fugas).

## Materiales necesarios

- **Materiales esenciales**: **[Vidrio](/es/materials/glass)** o plástico transparente (para el cilindro), **[Acero](/es/materials/steel)** inoxidable (para la aguja) y **[Caucho natural](/es/materials/natural-rubber)** (para la junta del émbolo).
- **Sustitutos posibles**: Se puede usar **[Cuero](/es/materials/leather)** para el sello del émbolo en versiones primitivas.

## Variantes y mejoras

- **Jeringas de vidrio**: Reutilizables y pueden esterilizarse hirviéndolas o en autoclave.
- **Agujas hipodérmicas**: Agujas huecas de acero utilizadas para inyecciones bajo la piel.
- **Luer-Lock**: Un sistema de conexión de tornillo estandarizado para evitar que las agujas se desprendan bajo presión.
- **Autoinyectores**: Dispositivos con resorte para la autoadministración rápida o de emergencia.

## Límites y riesgos

- **Esterilidad**: El factor más importante. Las jeringas no estériles pueden transmitir enfermedades mortales (como sepsis, VIH o hepatitis).
- **Embolia gaseosa**: Inyectar aire en el torrente sanguíneo puede ser fatal.
- **Fragility**: Las jeringas de vidrio son propensas a romperse y causar lesiones.
- **Precisión**: La fabricación debe ser precisa para asegurar una dosificación exacta.
