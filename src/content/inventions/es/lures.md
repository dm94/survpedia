# Breve descripción

Un señuelo de pesca es un atrayente artificial que provoca el ataque al imitar movimiento, brillo o silueta de una presa. Sustituye o complementa el cebo natural en la línea.

## Uso / Función

- **Atracción**: Provoca curiosidad o respuesta de alimentación.
- **Selección**: El tamaño y perfil ayudan a elegir especies.
- **Eficiencia**: Es reutilizable y reduce la necesidad de cebo fresco.

## Principio de funcionamiento

El señuelo genera estímulos como vibración, oscilación o destellos que los peces interpretan como presa. Con un **[Anzuelo de pesca](/es/inventions/fishing-hook)**, el pique fija el anzuelo al aplicar tensión en la línea.

## Cómo crearlo

1. **Dar forma al cuerpo**: Tallar **[Madera](/es/materials/wood)** o **[Hueso](/es/materials/bone)** en un perfil de pez o insecto.
2. **Balancear el cuerpo**: Ajustar el grosor y tallar un vientre ligeramente cóncavo para que vibre.
3. **Fijar el anzuelo**: Amarrar o alambar un anzuelo de **[Hierro](/es/materials/iron)** al cuerpo.
4. **Crear movimiento**: Tallar un labio o cara inclinada para que descienda y oscile.
5. **Sellar y proteger**: Recubrir con **[Resina](/es/materials/resin)** para reducir la absorción de agua.

## Materiales necesarios

- **Esenciales**: Cuerpo de madera o hueso, anzuelo de hierro.
- **Amarres**: **[Fibras vegetales](/es/materials/plant-fibers)** para sujetar.
- **Herramientas**: Cuchillo, punzón, lima, taladro pequeño, fuego para endurecer.
- **Sustitutos**: Espina o punta de hueso como anzuelo, concha tallada como cuerpo.

## Variantes y mejoras

- **Flotante**: Cuerpo ligero para superficie.
- **Hundido**: Cuerpo denso para mayor profundidad.
- **Con paleta**: Labio ancho que aumenta el bamboleo.
- **Con fibras**: Hilos o plumas para imitar insectos.

## Límites y riesgos

- **Enganches**: Se atrapa en troncos o algas.
- **Durabilidad**: La madera se hincha si no se sella.
- **Eficacia variable**: Algunos peces ignoran señuelos sin olor.
