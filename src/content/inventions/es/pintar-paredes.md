# Breve descripción

Pintar paredes es el proceso de preparar una superficie y aplicar pintura para sellarla, protegerla y darle color. Combina preparación del soporte (enlucidos, rellenos o encalado) con una aplicación controlada mediante pinceles para lograr un acabado uniforme y duradero.

## Uso / Función

- **Uso principal**: Proteger y colorear paredes interiores o exteriores.
- **Usos secundarios**: Reflejar luz, mejorar limpieza y marcar zonas o señales.
- **Escala**: Doméstica a edificios grandes.

## Principio de funcionamiento

La pintura forma una película fina que se adhiere a la pared. Las paredes porosas absorben humedad y aglutinante; la imprimación o el enlucido equilibran la absorción para que el secado sea uniforme. Varias capas finas mejoran la adhesión y reducen descamación.

## Cómo crearlo

### Versión funcional mínima

1.  Limpiar polvo y material suelto de la pared.
2.  Mezclar **[pintura](/es/materials/pintura)** con **[agua](/es/materials/water)** si es necesario.
3.  Aplicar una capa uniforme con **[pincel](/es/inventions/pincel)**.
4.  Dejar secar completamente antes de usar el espacio.

### Pasos esenciales

1.  **Reparar y limpiar**: Rellenar grietas con **[mortero](/es/materials/mortero)** o **[yeso](/es/materials/gypsum)** y retirar superficies que se desprenden.
2.  **Nivelar y sellar**: Aplicar una mano de **[cal](/es/materials/cal)** o enlucido fino para reducir la absorción.
3.  **Preparar la pintura**: Ajustar la viscosidad para que se extienda sin chorrear.
4.  **Recortar bordes**: Pintar esquinas y perímetros con pincel.
5.  **Cubrir la superficie**: Aplicar la capa principal con pasadas constantes.
6.  **Segunda capa**: Aplicar tras el secado completo para uniformidad y resistencia.

## Materiales necesarios

- **Materiales esenciales**:
  - **[Pintura](/es/materials/pintura)** o **[pigmentos](/es/materials/pigmentos)** con aglutinante
  - **[Agua](/es/materials/water)**
  - **[Yeso](/es/materials/gypsum)** o **[mortero](/es/materials/mortero)** para nivelar
  - **[Cal](/es/materials/cal)** para encalado o sellado
  - **[Arena](/es/materials/arena)** para rellenos o revoques
  - **[Ladrillo](/es/materials/ladrillo)** o **[madera](/es/materials/wood)** como soporte
- **Herramientas**: **[Pincel](/es/inventions/pincel)**, cubo de mezcla, palo de agitar, llana, trapos.
- **Posibles sustitutos**:
  - **Arcilla o barro coloreado**: Coloración temporal de paredes.
  - **Encalado**: **[cal](/es/materials/cal)** y agua para un acabado transpirable y barato.

## Variantes y mejoras

- **Encalado**: Acabado transpirable para mampostería.
- **Fresco**: Pigmento aplicado sobre yeso de **[cal](/es/materials/cal)** húmedo.
- **Temple a la cola**: **[Pintura](/es/materials/pintura)** con aglutinante animal para interiores.
- **Acrílicos modernos**: Durables, de secado rápido y lavables.

## Límites y riesgos

- **Descamación**: Preparación deficiente o humedad detrás del muro.
- **Agrietamiento**: Capas muy gruesas o enlucidos inestables.
- **Moho**: Humedad atrapada en espacios mal ventilados.
- **Toxicidad**: Vapores de solventes y pigmentos peligrosos requieren ventilación.
- **Caídas y polvo**: Escaleras, lijado y raspado implican riesgos.
