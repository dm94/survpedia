# Breve descripción

Un bisturí es un instrumento de hoja pequeña y extremadamente afilada utilizado para cirugía, disección anatómica y diversas artes y oficios.

## Uso / Función

- **Incisiones**: Realizar cortes precisos en piel y tejido.
- **Disección**: Separar tejidos.

## Principio de funcionamiento

Aplica alta presión en un área muy pequeña (el filo) para cortar los enlaces moleculares del material que se está cortando.

## Cómo crearlo

- **Moderno**: Hoja de acero desechable unida a un mango reutilizable.
- **Primitivo**: Tallado de obsidiana o pedernal hasta obtener un filo ultra afilado (más afilado que el acero pero frágil).

## Materiales necesarios

- **Acero de alto carbono**: Para la hoja (mantiene bien el filo).
- **Acero inoxidable**: Para el mango (resistencia a la corrosión).
- **Obsidiana**: Para hojas primitivas ultra afiladas.

## Variantes y mejoras

- **Desechable**: Hojas preesterilizadas de un solo uso.
- **Láser**: Utiliza luz para cortar y cauterizar simultáneamente.
- **Obsidiana**: Se utiliza en algunas cirugías modernas para minimizar las cicatrices debido a su filo a nivel atómico.

## Límites y riesgos

- **Desafilado**: El acero se desafila relativamente rápido en comparación con el diamante, pero es fácil de reafilar o reemplazar.
- **Lesión**: Extremadamente peligroso si se manipula incorrectamente.
