# Breve descripción

Una aguja quirúrgica es un instrumento delgado y afilado que se utiliza para pasar material de sutura a través del tejido para cerrar una herida o incisión.

## Uso / Función

Su función principal es perforar el tejido y guiar el hilo.
- **Sutura**: Unir tejidos.
- **Inyección**: (Agujas huecas) Administrar fluidos (no este tipo). Esto se refiere a agujas de sutura sólidas.

## Principio de funcionamiento

Concentra la fuerza en una punta afilada para penetrar el tejido con un trauma mínimo, creando un canal para que el hilo lo siga.

## Cómo crearla

- **Hueso**: Dando forma a una astilla de hueso y perforando un ojo.
- **Metal**: Estirando alambre de acero, afilando un extremo y aplanando/perforando el otro para el ojo (o engarzando el hilo directamente).

## Materiales necesarios

- **Acero**: Para agujas modernas, afiladas y duraderas.
- **Hueso/Madera**: Para versiones primitivas.

## Variantes y mejoras

- **Recta**: Se sostiene con la mano.
- **Curva**: Se usa con un portaagujas para espacios profundos o confinados.
- **Engarzada**: El hilo está unido permanentemente (sin ojo), causando menos trauma tisular.

## Límites y riesgos

- **Rotura**: Si se dobla demasiado.
- **Aguja perdida**: Un riesgo mayor en cirugía; puede migrar en el cuerpo.
