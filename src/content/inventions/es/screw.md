# Breve descripción

Un tornillo es una máquina simple que consiste en un plano inclinado enrollado alrededor de un cilindro o cono. Se utiliza principalmente para sujetar objetos o para levantar materiales.

## Uso / Función

- **Fijación:** Unir dos o más objetos de forma segura (ej. tornillos para madera, pernos).
- **Conversión de movimiento:** Convertir movimiento rotacional en movimiento lineal (ej. husillos en maquinaria).
- **Elevación:** Levantar agua o materiales granulares (ej. tornillo de Arquímedes).
- **Prensado:** Aplicar gran fuerza para exprimir o aplastar (ej. prensa de vino, imprenta).
- **Escala:** Desde pequeños componentes de relojería hasta prensas industriales masivas.

## Principio de funcionamiento

El tornillo funciona convirtiendo el par (fuerza rotacional) en fuerza lineal. La distancia entre las roscas se llama "paso".

- **Ventaja mecánica:** Una pequeña fuerza rotacional aplicada sobre una larga distancia (la circunferencia de la vuelta) crea una gran fuerza axial sobre una corta distancia (el paso).
- **Autobloqueo:** En muchos diseños, la fricción evita que el tornillo gire hacia atrás bajo carga, manteniéndolo seguro.

## Cómo crearlo

1.  **Preparación:** Seleccionar una varilla cilíndrica de madera o metal.
2.  **Marcado:** Dibujar una línea diagonal alrededor del cilindro para definir la trayectoria de la rosca (hélice).
3.  **Corte:**
    - **Madera:** Tallar el material entre las líneas marcadas para crear el perfil de la rosca.
    - **Metal:** Usar una lima o una herramienta de corte más dura (terraja) para cortar las roscas.
4.  **Aparejamiento:** Crear una rosca interna correspondiente (tuerca) roscando un agujero o tallando el interior de un cilindro hueco.

## Materiales necesarios

- **Varilla:** Madera dura (roble, nogal) o metal (bronce, hierro, acero).
- **Herramientas de corte:** Cinceles, limas o cortadores de metal más duro.
- **Lubricante:** Sebo o aceite para facilitar el corte y el movimiento.

## Variantes y mejoras

- **Tornillo de Arquímedes:** Un gran tornillo dentro de un tubo usado para levantar agua.
- **Tornillo para madera:** Punta cónica para penetrar la madera.
- **Perno:** Cilindro uniforme para uso con una tuerca.
- **Husillo de bolas:** Usa rodamientos de bolas para baja fricción (moderno de alta precisión).

## Límites y riesgos

- **Desgarro (Stripping):** Aplicar demasiado par puede arrancar las roscas, inutilizando el tornillo.
- **Fricción:** Una cantidad significativa de energía de entrada se pierde por fricción.
- **Corrosión:** Los tornillos de metal pueden oxidarse y agarrotarse, haciéndolos imposibles de remover.
