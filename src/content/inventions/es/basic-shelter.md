# Breve descripción

Un refugio básico es una estructura diseñada para proteger a los seres humanos de las inclemencias del tiempo, depredadores y amenazas externas. Combina una estructura de soporte con capas de aislamiento.

## Uso / Función

- **Protección térmica:** Mantener el calor corporal y proteger contra el viento, la lluvia o el sol.
- **Seguridad:** Espacio delimitado contra animales o intrusos.
- **Almacenamiento:** Lugar seco para guardar comida y herramientas.
- **Escala:** Doméstica (individual o familiar).

## Principio de funcionamiento

Utiliza materiales con baja conductividad térmica para el aislamiento y materiales rígidos para la integridad estructural. El diseño suele incluir una pendiente para el drenaje del agua y un espacio cerrado para atrapar aire estático (que actúa como aislante).

## Cómo crearlo

1. **Estructura:** Clavar postes de madera en el suelo o apoyarlos entre sí (forma de tipi o cobertizo).
2. **Cubierta:** Entrelazar ramas, colocar pieles o aplicar capas de paja/hojas sobre la estructura.
3. **Aislamiento/Sellado:** Aplicar barro en las grietas para evitar corrientes de aire o añadir capas gruesas de material vegetal.
4. **Nivel técnico:** Básico.

## Materiales necesarios

- **Esenciales:** Troncos o ramas gruesas para la estructura, follaje, pieles o corteza para la cubierta.
- **Herramientas:** Piedras para golpear, cuerdas para atar.
- **Sustitutos:** Cuevas naturales, nieve (iglú), piedras apiladas.

## Variantes y mejoras

- **Choza de ramas:** Muy rápida de construir, protección mínima.
- **Tienda de pieles:** Portátil y excelente aislamiento.
- **Cabaña de troncos/barro:** Permanente, muy resistente y térmicamente eficiente.

## Límites y riesgos

- **Incendio:** Los materiales suelen ser muy inflamables.
- **Colapso:** Una mala estructura puede caer por peso de nieve o viento fuerte.
- **Humedad:** Si no hay buena ventilación, puede acumularse moho o condensación.
