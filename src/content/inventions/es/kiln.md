# Breve descripción

Un horno es una cámara térmicamente aislada utilizada para calentar, hornear o secar sustancias. Es esencial para la cocina, la alfarería y la metalurgia.

## Uso / Función

- **Preparación de alimentos:** Hornear pan y cocinar alimentos.
- **[Alfarería](/es/inventions/pottery):** Cocer [arcilla](/es/materials/clay) para crear [cerámica](/es/materials/ceramics).
- **Metalurgia:** Fundir minerales y metales.
- **Fabricación:** Crear [ladrillos](/es/materials/brick) y [cal](/es/materials/lime).

## Principio de funcionamiento

El horno funciona confinando el calor dentro de un espacio aislado. Al controlar el flujo de aire (oxígeno) y combustible, se pueden alcanzar y mantener temperaturas muy altas, superando con creces las de un fuego abierto.

## Cómo crearlo

1.  **Cimientos:** Crear una base estable de piedra o tierra.
2.  **Cámara:** Construir paredes utilizando materiales resistentes al calor como ladrillos de arcilla o piedra.
3.  **Aislamiento:** Recubrir el exterior con capas gruesas de barro o arcilla para evitar la pérdida de calor.
4.  **Flujo de aire:** Incorporar una entrada de aire (se pueden usar fuelles) y una salida (chimenea).
5.  **Carga:** Crear una abertura para colocar artículos y combustible, que se pueda sellar durante la operación.

## Materiales necesarios

- **Estructura:** [Arcilla](/es/materials/clay), barro, ladrillos refractarios o [piedras](/es/materials/stone) resistentes al calor.
- **Unión:** [Arcilla](/es/materials/clay) húmeda o [mortero](/es/materials/mortar).
- **Combustible:** [Madera](/es/materials/wood), [carbón vegetal](/es/materials/charcoal) o [carbón mineral](/es/materials/coal).
- **Herramientas:** Palas, tenazas y potencialmente fuelles para temperaturas más altas.

## Variantes y mejoras

- **Horno de tierra:** Un simple hoyo en el suelo.
- **Horno de ladrillo:** Estructura duradera para una cocción consistente.
- **Horno de tiro ascendente:** El calor fluye desde el fondo a través de los artículos y sale por la parte superior.
- **Alto horno:** Utiliza aire forzado para alcanzar temperaturas lo suficientemente altas para la fundición de hierro.

## Límites y riesgos

- **Fallo estructural:** El calor extremo puede agrietar o derrumbar las paredes.
- **Humos tóxicos:** El monóxido de carbono y otros gases pueden acumularse en áreas mal ventiladas.
- **Peligro de incendio:** Las altas temperaturas pueden encender materiales inflamables cercanos.
- **Control de temperatura:** Difícil de mantener temperaturas precisas sin sensores modernos.

### Inventos relacionados

- **[Alfarería](/es/inventions/pottery):** Requiere un horno para la cocción.
- **[Chimenea](/es/inventions/chimney):** Esencial para el tiro de aire.
- **[Fuelles](/es/inventions/bellows):** Utilizados para aumentar la temperatura (si están disponibles).
- **[Cocción](/es/inventions/firing):** El proceso realizado en un horno.

### Materiales relacionados

- **[Carbón vegetal](/es/materials/charcoal):** Combustible común de alto calor.
- **[Arcilla](/es/materials/clay):** El material principal cocido en hornos.
- **[Cerámica](/es/materials/ceramics):** El producto de la cocción de arcilla.
- **[Ladrillo](/es/materials/brick):** Material de construcción cocido.
