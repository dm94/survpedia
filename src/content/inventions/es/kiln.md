# Breve descripción

Un horno es una cámara térmicamente aislada utilizada para calentar, cocer o secar sustancias. Es esencial para la cocina, la alfarería y la metalurgia.

## Uso / Función

- **Preparación de alimentos:** Cocer pan y cocinar alimentos.
- **Alfarería:** Cocer arcilla para crear cerámica.
- **Metalurgia:** Fundir minerales y metales.
- **Fabricación:** Creación de ladrillos y cal.

## Principio de funcionamiento

El horno funciona confinando el calor dentro de un espacio aislado. Al controlar el flujo de aire (oxígeno) y el combustible, se pueden alcanzar y mantener temperaturas muy altas, que superan con creces las de una hoguera abierta.

## Cómo crearlo

1.  **Cimentación:** Crear una base estable de piedra o tierra.
2.  **Cámara:** Construir paredes utilizando materiales resistentes al calor como ladrillos de arcilla o piedra.
3.  **Aislamiento:** Revestir el exterior con capas gruesas de barro o arcilla para evitar la pérdida de calor.
4.  **Flujo de aire:** Incorporar una entrada de aire (se pueden usar fuelles) y una salida (chimenea).
5.  **Carga:** Crear una abertura para colocar los objetos y el combustible, que pueda sellarse durante el funcionamiento.

## Materiales necesarios

- **Estructura:** Arcilla, barro, ladrillos refractarios o piedras resistentes al calor.
- **Ligante:** Arcilla húmeda o mortero.
- **Combustible:** Madera, carbón vegetal o carbón mineral.
- **Herramientas:** Palas, pinzas y potencialmente fuelles para temperaturas más altas.

## Variantes y mejoras

- **Horno de tierra:** Un simple foso en el suelo.
- **Horno de ladrillo:** Estructura duradera para una cocción constante.
- **Horno de tiro ascendente:** El calor fluye desde la base a través de los objetos y sale por la parte superior.
- **Alto horno:** Utiliza aire forzado para alcanzar temperaturas lo suficientemente altas como para fundir hierro.

## Límites y riesgos

- **Fallo estructural:** El calor extremo puede agrietar o colapsar las paredes.
- **Gases tóxicos:** El monóxido de carbono y otros gases pueden acumularse en zonas mal ventiladas.
- **Riesgo de incendio:** Las altas temperaturas pueden inflamar materiales inflamables cercanos.
- **Control de temperatura:** Difícil de mantener temperaturas precisas sin sensores modernos.
