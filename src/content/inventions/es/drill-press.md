# Breve descripción

Un taladro de columna es una máquina herramienta estacionaria utilizada para perforar agujeros precisos en madera, metal o plástico. Consta de una base, una columna, una mesa y un cabezal de potencia con un husillo giratorio. Ofrece mucha mayor precisión y potencia que un taladro manual.

## Uso / Función

- **Uso principal:** Perforar agujeros perfectamente perpendiculares a la superficie.
- **Usos secundarios:** Escariado, roscado, avellanado y lijado (con accesorios de tambor).
- **Escala:** Taller/Industrial.

## Principio de funcionamiento

Un motor impulsa un husillo a través de un sistema de correas y poleas (para cambiar velocidades). El husillo sostiene un portabrocas o cono para la broca. Una palanca de avance (caña) mueve el husillo giratorio verticalmente hacia la pieza de trabajo, que se sujeta o sostiene en una mesa ajustable.

## Cómo crearlo

### Versión funcional mínima (Taladro de Poste)

1.  **Bastidor:** Un poste de madera o metal resistente atornillado a una pared o banco.
2.  **Husillo:** Un eje vertical con una manivela o volante para impulso.
3.  **Avance:** Un mecanismo de tornillo para hacer avanzar la broca mientras gira (avance automático) o una palanca simple.

### Versión intermedia (Impulsado por Correa)

1.  **Columna:** Un tubo de acero rígido o columna de hierro fundido.
2.  **Cabezal:** Contiene los cojinetes del husillo y el conjunto de la caña.
3.  **Impulso:** Un sistema de poleas escalonadas para permitir diferentes velocidades (lentas para metal, rápidas para madera).
4.  **Mesa:** Puede elevarse, bajarse y girar alrededor de la columna.

## Materiales necesarios

- **Esenciales:**
  - **Hierro fundido:** Para el cabezal, la mesa y la base (rigidez).
  - **Acero:** Para la columna, el husillo y la caña.
  - **Cojinetes:** Bronce o rodamientos de bolas para una rotación suave.
- **Herramientas:** **[Torno](/es/inventions/lathe)** (para el husillo), **[Limas](/es/inventions/file)**, **[Broca](/es/inventions/drill-bit)**.

## Variantes y mejoras

- **Taladro de Banco:** Más pequeño, se monta sobre un banco de trabajo.
- **Taladro de Pie:** Columna más alta, se apoya en el suelo para trabajos más grandes.
- **Taladro Radial:** El cabezal puede moverse a lo largo de un brazo para perforar placas grandes sin mover la pieza.
- **Taladro Magnético:** Taladro portátil que se sujeta a estructuras de acero con un electroimán.

## Límites y riesgos

- **Sujeción de la pieza:** Si la broca agarra la pieza, puede girar peligrosamente ("efecto helicóptero"). Siempre sujete la pieza.
- **Velocidad:** La velocidad incorrecta quema las brocas (demasiado rápido en acero) o desgarra la madera (demasiado lento).
- **Cabello/Ropa:** El husillo giratorio es un peligro mayor de enredo.
