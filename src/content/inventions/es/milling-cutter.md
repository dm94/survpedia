# Breve descripción

Una fresa es una herramienta de corte rotativa utilizada en fresadoras para eliminar material de una pieza de trabajo. A diferencia de una broca, que corta solo en la punta, una fresa puede cortar con sus lados, permitiendo el mecanizado de superficies planas, ranuras y contornos complejos.

## Uso / Función

- **Mecanizado de Superficies**: Aplanar bloques de material (planeado).
- **Ranurado**: Cortar canales o chaveteros (fresado de ranuras).
- **Contorneado**: Crear formas complejas siguiendo un perfil.
- **Escala**: Desde micro-mecanizado hasta grandes fresadoras industriales.

## Principio de funcionamiento

La fresa gira a alta velocidad y tiene múltiples bordes cortantes (dientes o labios). A medida que la pieza de trabajo se alimenta contra la fresa giratoria, cada diente arranca una pequeña viruta de material.

- **Corte periférico**: Los dientes en la circunferencia cortan mientras la herramienta se mueve lateralmente.
- **Corte frontal**: Los dientes en el extremo cortan cuando la herramienta se hunde o planea.

## Cómo crearla

### Versión funcional mínima

Una barra de acero endurecido con dientes tallados a mano (con limas antes del endurecimiento) puede funcionar para materiales blandos como madera o plástico.

### Versión avanzada (Acero Rápido - HSS)

1.  **Torneado**: Se tornea una barra de acero de alta velocidad a la forma básica.
2.  **Fresado de estrías**: Se utiliza otra fresadora (o una máquina divisora) para cortar las ranuras helicoidales que forman los bordes de corte.
3.  **Endurecimiento**: El acero se calienta y se templa para lograr la máxima dureza sin ser demasiado frágil.
4.  **Afilado**: Se rectifican los bordes de corte con precisión usando muelas abrasivas.

## Materiales necesarios

- **Esenciales**:
  - **Acero de Alta Velocidad (HSS)**: Aleación de acero con tungsteno, cromo y vanadio que mantiene su dureza a altas temperaturas.
  - **Carburo de Tungsteno**: Para herramientas modernas de alto rendimiento, más duro pero más frágil.
- **Herramientas**: **[Torno](/es/inventions/lathe)**, **[Fresadora](/es/inventions/milling-machine)** (para hacer fresas complejas), **[Muela de esmeril](/es/inventions/grinding-wheel)**.

## Variantes y mejoras

- **Fresa de dedo (End Mill)**: Corta con el extremo y los lados; la más común para ranuras y perfiles.
- **Fresa de planear (Face Mill)**: Gran diámetro, para aplanar superficies grandes rápidamente.
- **Fresa de bola**: Punta redondeada para mecanizado 3D y superficies curvas.
- **Fresa de forma**: Perfil personalizado (ej. para cortar engranajes).

## Límites y riesgos

- **Desgaste**: Los bordes se desafilan con el uso, aumentando la fricción y el calor.
- **Deflexión**: Si la fresa es larga y delgada, puede doblarse bajo carga, causando imprecisiones.
- **Rotura**: Si se alimenta demasiado rápido o la fresa se atasca, puede romperse violentamente.
- **Calor**: El corte genera mucho calor; a menudo se requiere refrigerante.
