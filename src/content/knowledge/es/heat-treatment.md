# Tratamiento Térmico

El tratamiento térmico implica el uso de calentamiento o enfriamiento, normalmente a temperaturas extremas, para lograr un resultado deseado como el endurecimiento o ablandamiento de un material. Se aplica principalmente a los metales y se utiliza para alterar sus propiedades físicas y, a veces, químicas.

## Procesos Comunes

### 1. Recocido (Annealing)
Calentar el metal a una temperatura específica y luego dejarlo enfriar lentamente (a menudo dentro de la fragua apagándose). Esto ablanda el metal, lo hace más trabajable (maleable) y alivia las tensiones internas.
- **Uso:** Cuando una pieza se vuelve demasiado dura para trabajar durante la forja, se recuece para evitar grietas.

### 2. Normalizado
Calentar el metal por encima de una temperatura crítica y luego enfriarlo al aire libre. Esto refina la estructura del grano y mejora la uniformidad.
- **Uso:** Para devolver el metal a un estado "normal" después de la forja pero antes del endurecimiento.

### 3. Templado (Hardening/Quenching)
Calentar el metal a una temperatura crítica (donde se vuelve no magnético en el caso del acero) y luego enfriarlo rápidamente sumergiéndolo en agua, aceite o salmuera. Esto hace que el metal sea extremadamente duro pero también frágil.
- **Crítico:** Solo el acero con suficiente carbono puede endurecerse. El hierro no.

### 4. Revenido (Tempering)
Se realiza después del templado. El metal se vuelve a calentar a una temperatura más baja (a menudo juzgada por el color de la capa de óxido formada, por ejemplo, paja, púrpura, azul) y luego se enfría. Esto sacrifica algo de dureza para aumentar la tenacidad y reducir la fragilidad.
- **Esencial:** Una hoja templada sin revenir se romperá como el vidrio al impactar.

### 5. Cementación (Case Hardening)
Utilizado para hierro o acero con bajo contenido de carbono. La superficie exterior se infunde con carbono (utilizando hueso, cuero o carbón) y luego se endurece, dejando un núcleo tenaz y dúctil.

## Medios de Enfriamiento

- **[Agua](/es/materials/water):** Enfriamiento rápido. Máxima dureza, alto riesgo de grietas.
- **[Aceite Vegetal](/es/materials/vegetable-oil):** Enfriamiento más lento. Buena dureza, menor riesgo de grietas.
- **Salmuera (Agua Salada):** Enfriamiento muy rápido. Usado para aceros específicos.
- **Aire:** Enfriamiento lento. Usado para normalizado o aceros de endurecimiento al aire.
