# Contributing to Survpedia

Thank you for your interest in contributing to Survpedia! We welcome contributions to help expand our technological survival manual codex.

## How to Contribute

1.  **Fork the repository** to your GitHub account.
2.  **Clone the repository** to your local machine.
3.  **Create a new branch** for your contribution (e.g., `feat/add-new-element`).
4.  **Make your changes**.
5.  **Commit your changes** with a descriptive commit message.
6.  **Push your branch** to your fork.
7.  **Open a Pull Request** to the main repository.

## Adding New Content

Survpedia uses Markdown files with Frontmatter to store content. To add new content, please use the templates below as a guide.

### 1. Elements

To add a new element, create a new Markdown file in `src/content/elements/[lang]/` (e.g., `src/content/elements/en/gold.md`).

**Template:** [Element Template](src/content/_templates/en/element.md)

### 2. Inventions

To add a new invention, create a new Markdown file in `src/content/inventions/[lang]/` (e.g., `src/content/inventions/en/printing-press.md`).

**Template:** [Invention Template](src/content/_templates/en/invention.md)

### 3. Materials

To add a new material, create a new Markdown file in `src/content/materials/[lang]/` (e.g., `src/content/materials/en/steel.md`).

**Template:** [Material Template](src/content/_templates/en/material.md)

## Localization

Survpedia supports multiple languages. Currently, we support:

- English (`en`)
- Spanish (`es`)

If possible, please provide translations for your content in the corresponding directories.
